     |File area #35 . Apricot Network Update files (2.1}2.11/7.1}7.11)|
     +----------------------------------------------------------------+
 
 These files will form an update disk that will update an MF_NET Rel 7.1
 or VX_Net Rel 2.1 Network so that the new XEN_s machines are supported and
 can remote-boot. All the files should be copied onto one floppy disk, and
 the INSTALL batch file run from that disk.
