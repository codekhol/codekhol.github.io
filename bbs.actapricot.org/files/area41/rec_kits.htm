The Recovery Kit

The only purpose of the Recovery Kit is to restore the software on the hard disk of an MS Series computer to the state in which it left the factory. It cannot be used to install individual items of software.

The Recovery Kit will comprise of a Recovery CD and a Recovery Diskette, unless you have an MS600 computer with a product code (MVxxxxx-x), In which case it will comprise of just a Recovery CD.

Do not assume that full recovery is the best, or the only way to fix a problem with your PC. Problems can often be fixed with less drastic measures. Before you do anything, consult the chapter on 'Troubleshooting' in the Owner’s Handbook, and the documentation for Windows 95. Also, don't forget to check the documentation, which came with any software or hardware you have added to the computer.

Mitsubishi cannot be held responsible for any loss of data, caused by using the Recovery Kit.

The following lists show which Recovery kit (floppy and cd) can be used on a specific MS Series computer.

  5. MS660 (model MFxxxxx-x)
