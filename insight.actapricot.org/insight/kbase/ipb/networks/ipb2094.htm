LAN Manager 2.1 / OS/2 1.31 with 2 Adaptec AHA1640 cards in FTs and FTe

In LAN Manager Newsletter - Issue 2, published on 3rd June 1991, we documented the recommended settings for FT servers with two Adaptec SCSI controller cards that are being set up for disk duplexing under LAN Manager 2.00b (and OS/2 1.21).

With the release of LAN Manager 2.1 and OS/2 1.31, these recommended settings need to be altered. When booting from the OS/2 1.31 install disk, OS/2 1.31 will install onto the first disk attached to the Adaptec controller card located at base I/O address 330h, but when booting from the hard disk, OS/2 1.31 will try to load the operating system from the first hard disk attached to the Adaptec card in the highest numbered MCA slot.

This means that if a server is configured as per LAN Manager Newsletter Issue 2, and OS/2 1.31 is loaded onto it, the server will NOT boot from the hard disk. Also, if disk duplexing is configured, the primary partition for the mirrored pair will be located on the disk attached to the Adaptec card in the highest numbered MCA slot (which by default will be the disk mounted in the rear drive bay of a VX FT/FTs).

In order to correctly configure an FT server (VX FT, FTs or FTe) for OS/2 1.31 when two Adaptec controller cards are fitted, you should set the system up as follows...

For VX FT servers (FT mark 1), or FTs servers (FT mark 2)
Slot 1(Mk1): AHA-1640 SCSI Host Adapter (Attached to the drive(s) in the REAR BAY)
or Slot 7(Mk2):
Adapter BIOS BIOS Disabled
I/O Port Address 334h
Arbitration Level Level_7
Interrupt Level IRQ_15
Slot 2(Mk1): AHA-1640 SCSI Host Adapter (Attached to the drive(s) in the FRONT BAY)or Slot 8(Mk2):
Adapter BIOS DC000h The first disk on this controller
I/O Port Address 330h will be the OS/2 boot disk.
Arbitration Level Level_6
Interrupt Level IRQ_14
For FTe servers
Slot 2: AHA-1640 SCSI Host Adapter - (Attached to the drive(s) to be the SECONDARY disk(s))
Adapter BIOS BIOS Disabled
I/O Port Address 334h
Arbitration Level Level_7
Interrupt Level IRQ_15
Slot 3: AHA-1640 SCSI Host Adapter - (Attached to the drive(s) to be the PRIMARY disk(s))
Adapter BIOS DC000h The first disk on this controller
I/O Port Address 330h will be the OS/2 boot disk.
Arbitration Level Level_6
Interrupt Level IRQ_14
