Installing NetWare 3.12 and NetWare 4.01 from CD-ROM

At the time of writing this bulletin installation is only supported using BUSLOGIC SCSI adapters.

To install NetWare the CD-ROM drive must be able to be accessed as a DOS drive. Listed below are recommend settings that enable the CD-ROM to be accessed as a DOS device.

CONFIG.SYS Single SCSI Adapter
DEVICE=C:\BTDOSM.SYS /p330 /d /m
DEVICE=C:\BTCDROM.SYS /D:ROM

CONFIG.SYS with only a CD-ROM on second SCSI Adapter
DEVICE=C:\BTDOSM.SYS /p330 /d /m
DEVICE=C:\BTDOSM.SYS /p334 /i /d
DEVICE=C:\BTCDROM.SYS /D:ROM

CONFIG.SYS with CD-ROM and other Devices on second SCSI Adapter
DEVICE=C:\BTDOSM.SYS /p330 /d /m
DEVICE=C:\BTDOSM.SYS /p334 /i /d /m
DEVICE=C:\BTCDROM.SYS /D:ROM

AUTOEXEC.BAT
C:\MSCDEX.EXE /D:ROM /M:12 /l:d /v /e

Version BTDOSM.SYS = 2.00 or greater
Version BTCDROM.SYS = 3.00i or greater
Version MSCDEX.EXE = 2.22 or greater

CD-ROM can now be accessed as drive D.

Note: /p??? IO address of BUSLOGIC Adapter
/D:??? Name given to CD-ROM device. Do not use CDROM as device name.
/l:? DOS drive letter for CD-ROM drive

NetWare 3.12

When you load the BUSLOGIC disk driver please exclude the SCSI ID of the CD-ROM drive.

i.e. LOAD BT312 PORT=330 EXCLUDE_IDS=6

NetWare 4.01

When NetWare 4.x install program prompts for you to enter "Disk Driver Type". Select BT40 and change option "SCSI targets to hide from NetWare" enter the SCSI id of the CD-ROM. This will then invoke the correct parameters to load the BUSLOGIC disk driver.

i.e. LOAD BT40 PORT=330 EXCLUDE_IDS=6

After the installation is complete please remove all DOS drivers that relate to the CD-ROM drive.

Note: If the CD-ROM is going to be used as a NetWare Volume do not remove the parameter "EXCLUDE_IDS = ?"

All necessary files can be found on Apricot Insight BBS in Area 37. File name NETINSCD.ZIP
