Novell NetWare Updates

A number of new NetWare updates have just been released by Novell. All the updates documented below can be found in File Area 37 of Apricot Insight BBS.

  • LIBUP3.EXE
    This update contains the latest CLIB, MATHLIB, MATHLIBC and AFTER311 modules for NetWare v3.11, SFT III v3.11, v3.12 and v4.01. The update includes fixes for the problems found in the CLIB.NLM that was shipped in LIBUP2.EXE, so coexistence with added name space support, and coexistence with NMS has been fixed for NetWare v3.11 servers.

    The update includes full documentation, and an install routine to simplify the installation of the files onto live servers.

  • 401PT1.EXE
    This update contains a number of updates for NetWare v4.01, including LAN drivers, SBACKUP files, PSERVER files, STREAMS updates, PBURST updates, Router updates and an update to Directory Services. All the updates in this archive are available separately, but Novell have grouped them all together in a single source update for easy reference. This update should be used in conjunction with 401PT2.EXE, which contains Dynamic Operating System patches for NetWare v4.01.

  • 312PT1.EXE
    This update contains a number of Dynamic Operating System patches for NetWare v3.12. There are currently 20 patches in the NetWare v3.12 patch kit, and all are documented in the file 312PT1.TXT that is contained within the archive. The kit includes fixes for TTS, disk space reporting errors, access denied errors, extended attributes for OS/2 clients, and hanging servers when re-mirroring under heavy load.

All the above updates contain full documentation detailing the changes and fixes that have been made, along with full installation instructions. This documentation should be read and understood prior to installation. Apricot recommend that these updates be installed on ALL relevant servers.
