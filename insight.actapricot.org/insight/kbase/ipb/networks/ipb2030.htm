Network Drivers Disk Release 3

Further to IPB 2029, Network Drivers Disk Release 3 has now been released.

The disk contains drivers for use with the on-board Ethernet adapter, as found on the XEN-S, Qi, FT and LANstation range, for use on Novell (2.0a, 2.1x and 386), DEC-Net DOS, LAN Manager (and other networks that use an NDIS/MAC driver), and PC-NFS. The disk also contains a special Apricot DMA driver that should be used when installing versions of Windows 3 that were NOT supplied by Apricot if the on-board Ethernet adapter is to be used.

This disk will be supplied in-the-box with all new equipment shipped. The files that make up the disk are also available for download from file area 37 of the Apricot Technical Support Bulletin Board. The download file is called NETDRIV3.EXE, is 96477 bytes long, and is a self-unpacking archive that contains ALL the drivers on the disk.
