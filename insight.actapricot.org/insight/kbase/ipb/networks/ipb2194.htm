LS Pro (Caracal) Limitations

There are a number of features and issues with the initial shipments of the new LS Pro that customers should be aware of. Some are issues that will be resolved in future BIOS releases (the BIOS can be flash upgraded), others are configuration issues that should be noted to prevent possible support calls. This IPB applies to LS Pro (Caracal) BIOS version 3.04.

  1. With the initial release of the LS Pro, the Power Management facility is not available, however Hard Disk and Monitor power down functions are still available. This means that Energy Star compliance cannot be guaranteed. Full power management facilities will be available at a later date.

  2. If you enable the power-on sound and set the volume to maximum, when you re-enter SETUP, the volume slider indicating the volume level will be slightly below maximum.

  3. If the on-board AMD Ethernet is disabled in SETUP, the machine must be powered off and back on after saving the configuration for the change to take effect - warm-booting the system will not change the status.

  4. It is not currently possible to disable Floppy Boot in SETUP (it is possible however to fully disable the Floppy Disk Drive).

  5. It is not currently possible to disable the Hard Disk Drive in SETUP.

  6. It is not currently possible to disable the Parallel Port in SETUP.

  7. In the LOC security user setup screen, it is not possible to define more than one login period per day for a user (the block can be any size, but only one block can be defined).

  8. If LOC security is used, it is possible to enter non-alpha-numeric characters when setting up a user's username and/or password (the LS Pro manual states that only alpha-numeric characters or space are valid). These characters will then form part of the username and/or password. We would recommend however that only valid alpha-numeric characters and space are used, as when this feature is corrected, any users with invalid characters will be locked out.

  9. In LOC security, usernames are case sensitive, whereas passwords are not.

  10. In LOC security, if an invalid username or password is entered at login time, the username and password fields are not cleared when the user is asked to re-enter the login details.

  11. If installing NetWare or NT onto an LS Pro with an IDE disk greater than 500MB (the Quantum 540MB disk for example), make sure that the Large HD Translation option in the Advanced Options screen of SETUP is disabled.

  12. When installing the Phoenix PCM+ Card and Socket Services software (to enable the use of PCMCIA cards in the system), the software will default to using Interrupt 15. This should be changed to Interrupt 10. If the default value is left unchanged, the PCM+ software will not operate correctly.

  13. When installing the Crystal audio drivers for Windows, the settings to use are; Duplex FULL, I/O Addr 530H, DMA Channel Play 1, DMA Channel Capture 0, IRQ 11. It is also worth pointing out the Crystal audio is Microsoft Sound System compatible.
