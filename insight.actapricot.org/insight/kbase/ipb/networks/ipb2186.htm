Configuring NT to use the Sony SLCD CD-ROM on a XEN-LS II

The Sony SCLD CD-ROM driver can be downloaded from Apricot Insight BBS in file area 88, file name S31X86.EXE.

Install the SCSI Adapter: Sony CDU 31A from within Windows Setup and Add SCSI Adapters option.

NOTE:

Installing on a XEN-LS II (Rev F). Model Number "SF".

  • If installing the CD-ROM driver on a XEN-LS II (Revision F) motherboard, Model number prefix "SF", the following steps for Updating the System Registry should NOT be followed as the CD-ROM drive is automatically detected.

    • Installing on a XEN-LS II (Rev D). Model Number "SA".

  • If installing the CD-ROM driver on a XEN-LS II (Revision D) motherboard, Model number prefix "SA" the following steps for Updating the System Registry should be followed:-

    • Updating the System Registry

  • The System Registry should the be modified to change the enter the IRQ address and the Base I/O address. The following steps should be taken to modify the System Registry:-

    1. Run the program REGEDT32 to invoke the System Registry Editor.

    2. Select the HKEY_LOCAL_MACHINE on Local Machine section.

    3. Select SYSTEM, CurrentControlSet, Services, Slcd32

    4. Select Edit from the menu and select Add Key.

    5. Enter Key Name: Parameters and select OK.

    6. Select Parameters.

    7. Select Edit from the menu and select Add Key.

    8. Enter Key Name: Device0 and select OK.

    9. Select Device0

    10. Choose Edit from the menu and Add Value.

    11. Enter Value: DriverParameter, accept data type REG_SZ.

    12. Enter string: IRQ=5 BASE=320

    • Once the above has been completed, exit the System Registry and shutdown Windows NT. Windows NT is now configured to use the Sony SLCD CD-ROM drive.
