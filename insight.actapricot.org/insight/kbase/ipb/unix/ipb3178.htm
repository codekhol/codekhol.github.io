Getting UNIX files from Apricot

The following methods / areas are available to obtain files electronically from Apricot.

  1. Insight BBS

UNIX related files which can be downloaded are located in three areas:-

area27 Files which are available for public access
area39 Files (includes SCO Supplements) which are available to all Apricot sales channels and direct support customers.
area56 Chargeable or early release UNIX software (additional password required)
  1. UUCP from UNIX Server

The UNIX server can be reached on 0121-717-045[67], please ask if you wish to have an account added.

/uuinfo/bbs Link to Bulletin Board (see above)
/uuinfo/efs SCO Extended Feature Supplements
/uuinfo/lng SCO Language and old Network Supplements
/uuinfo/net SCO Network Supplements
/uuinfo/unx SCO UNIX Supplements
/uuinfo/uod SCO UNIX and Open Server Supplements
/uuinfo/xnx SCO Xenix Supplements
  1. Insight CD-ROM

Releases 4 and future releases contain SCO supplements to save download time and costs. The new release 5 UNIX files are located in:

\unix\images SCO Supplements
\unix\scripts Miscellaneous other supplements
\unix\utility Miscellaneous utilities.
  1. Internet

Apricot now has a server available on the Internet for file downloads. To access the server from your Internet machine please use the anonymous login and password procedure. The machine can be reached with the command:

ftp ftp.apricot.co.uk

Files are located in the following directories:

/pub/apricot Apricot specific files (usually temporary)
/pub/bbs Link to Bulletin Board (see above) but only public areas available (area27 only for UNIX)
/pub/netware NetWare related files
/pub/sco SCO related files
/pub/src Source (usually ‘C’)
/pub/unixware UnixWare related files
/pub/winnt Windows NT related files
/pub/windows Windows related files

Further Internet facilities are being developed and further information will be issued when these are ready.
