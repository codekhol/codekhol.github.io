SCO Advanced Hardware Supplement Release 3.4

See also IPB 3196.

This supplement from SCO contains four individual supplemental driver floppies. The present release (3.4) and the previous release (3.3) have both also been named by SCO as EFS140.

The four components of the AHS 3.4 are:

LLI Drivers Disk Rel 3.4
Advanced Graphics Adapter EFS Rel 3.4
BTLD Drivers EFS Rel 3.4
Multiprocessing Drivers EFS Rel 3.4

The following files are in area39 of Apricot Insight BBS and on release 5 of the Insight CD:

The files in the above table are in Unix compress format. Use the Unix command uncompress and, for the disk images, then use dd to copy the resulting file to a formatted floppy. For example:

#uncompress efs140lli.Z

#dd if=efs140lli of=/dev/fd0135ds18 bs=9k

LLI Drivers Disk Rel 3.4

This version contains all of the SCO supported Ethernet card drivers and is supported for use with TCP 1.2.1. Of specific importance to Apricot hardware is an improved 3Com Etherlink III driver for the 3C509, 3C529, and 3C579 cards which fixes a problem whereby the card could stop working on fast machines.

Advanced Graphics Adapter EFS Rel 3.4

This contains the latest video card X Window drivers for Open Server / Open Desktop 3.0. This AGA EFS contains Cirrus Logic drivers for the 5426, 5428, and 5434. When using the XEN-PC 590 with the on-board 5434 the 5434 driver should be used and will allow a resolution of 1024x768x256 colours at 70Hz with a suitable monitor, selecting a 64k colour driver will result in an error message when the Controls-->Colour icon is selected. A resolution of 1280x1024x256 colours at 60Hz will also function with a suitable monitor.

BTLD Drivers EFS Rel 3.4

This contains Boot Time Loadable drivers for various SCSI adapters supported by SCO. Whilst there is a driver for the Adaptec AIC-7770 on this supplement, we recommend using the BTLD driver shipped with the FT//ex for use with the on-board SCSI adapter on this machine at the present time.
