APC PowerChute Plus for SCO UNIX

These notes refer to Version 4.2.1 of PowerChute Plus for SCO UNIX only. The installation procedure is unusual in that the product does not use custom (ADM) to install, and is outlined below:

  1. With UNIX in multi-user mode log in as the Super User:
    root
  2. Insert the “Install Disk” and copy the INSTALL script into a temporary directory:
    cd /tmp
    tar xvf /dev/install INSTALL
  3. Run the INSTALL script:
    ./INSTALL
  4. When installation is complete remove the INSTALL script:
    rm INSTALL

Which serial device will be dedicated to PowerChute Plus for Unix [?] 1

The UPS would normally be connected to the first on-board serial port (/dev/tty1a - option 1 above). You should check that both the modem and non modem devices for the port you select are disabled (i.e. disable tty1a and disable tty1A at the end of the installation if you have not already done so).
