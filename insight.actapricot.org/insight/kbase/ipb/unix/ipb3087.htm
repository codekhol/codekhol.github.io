SCO UNIX V/386 3.2 Version 2

General Information

This product commenced shipment on Monday 10th December under the existing MCA Product Code of SS 43710. Where an upgrade is required from 3.2.0 which does not meet the requirements outlined in UNIX News 7, then the product can be purchased from Apricot for £125.00 RRP, the product code is SS46860 and will be available from 21st December.

An Upgrade to the SCO UNIX Development system will be available in January, product code is SS46861.

Installation

During installation a few minor issues may need to be resolved. Details are given below on corrections required.

  1. To install a second Adaptec 1640 SCSI adapter, alter the file /etc/conf/sdevice.d to
    ad Y 1 5 1 15 330 332 0 0
    ad Y 1 5 1 12 334 336 0 0

    then alter the ad line in /etc/conf/cf.d/mdevice to

    ad iI iHrZCc aha 34 34 1 2 5

    The remainder of the procedure can be achieved running mkdev hd twice to install the second hard disk.

  2. When installing TCP/IP a clash will occur with the wdha entry in the mdevice file mentioned above. Change the wdha line to:
    wdha iI iHoCc wdha 35 35 1 2 -1
  3. The values of some kernel parameters may be reduced by installation of some products (notably TCP/IP). We recommend that the following values are set as a minimum.
    NFILE 500 NINODE 500
    NS5INODE 500 NPROC 300
    NREGION 350 NBLK128 256
    NBLK256 256
  4. The installation will ask for a disk "Master", this is the disk labelled M1.
