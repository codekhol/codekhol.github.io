XEN-S Memory Upgrades

See Also
IPB 4022 - IPB 4027 - IPB 4044 - IPB 4046
IPB 4060 - IPB 4066 - IPB 4068 - IPB 4100

Some confusion is still evident regarding RAM daughterboard upgrades for XEN-S machines. The following cross reference chart is an attempt to clarify the situation.

Memory Module XEN-S (I) Venus I XEN-S (II) Venus II
1MB Daughterboard XB42028 or XB45478 XB42028 or XB45478
2MB Daughterboard XB42594 or XB44938 XB42594 or XB44938
4MB Daughterboard XB42567 or XB44939 XB42567 or XB44939
8MB Daughterboard N/A XB44940

Notes

  1. The alternative product code is that for the memory daughterboard for the new LANstation. The daughterboards are electrically the same but the LANstation board is a little smaller. This does has no affect when installing into XEN-S but XEN-S boards may not be installed into LANstation.

  2. The 8MB daughterboard will not fit into XEN-S (I) products because of the lack of sufficient connectors. XEN-S (II) products can be identified by the letter 'E' after the model number  on the base serial number label. {E.g. 386/50E}

  3. 2MB Daughterboards require a minimum ROM BIOS version of VR1.2.6. 1 and 4MB Daughterboards require a minimum ROM BIOS version of VR1.2.12. All daughterboards require a minimum SETUP.EXE version of VR1.2.6

  4. Version 1.2.6 and later versions of SETUP.EXE require that all the memory fitted into the machine be configured as base memory. E.g. If a 4MB daughterboard is fitted the total machine memory will be 5MB. Base memory in SETUP.EXE will therefore contain 5120 and Extended memory will contain 4096.
