MS 'Diamond' Remote Control (Rev 1.9.1)

Hand Held Transmitter

31 Remote controller keys

FSR (Force Sensitive Resistor) : displacement -44 to 44 in X and Y

2 x AA batteries (Alkaline or NiCd)

Range : 3m

Battery Life : 100 days (typical use) corresponds to 30 mins of continuous use per day @ 30mA. The life is considerably extended if little use is made of the FSR.

Auto power save

LED for IR emission indication

Custom coded

IR link is 100% compatible with the IR Multiple Protocol Specifications V0.2

 

Standby Puts the PC into standby mode. If already in standby mode the PC will switch on. When in standby mode the PC may also be reactivated by pressing any of the following keys: Message, CD, TV or Teletext
  
Message Starts the Message centre application for playing back / viewing Voice / Fax and E-mail messages.
  
CD Starts the Audio CD player.
  
TV Starts the TV application. Subsequent key presses will toggle the TV application into its different screen display modes (Window, Full Screen, Backdrop).
  
Teletext Starts the TV application in Teletext mode. Subsequent key presses will toggle the TV application display mode between TV, Teletext and TV / Teletext mix.
  
Mute Mutes or un-mutes all the PC audio sources.
  
Help Displays or hides the Remote Status window.
  
FastText These coloured buttons drive the FastText capabilities of the Teletext application. If the Remote Status window is being displayed then the buttons become software programmable.
  
Channel These buttons [+] and [-] increase / decrease the TV channel or track number dependant on the application that currently has the Remote focus as detailed below:
Application Effect
TV Tuner Pre-programmed channel number.
Teletext Page number.
CD Player Track number.
Wave Player Track number.
MIDI Player Track number.
Radio Pre-programmed channel number.
  
Volume Increase / decrease the volume of the individual applications. This does not affect the master volume of the system. The Remote Status window will appear while the Volume keys are depressed and will disappear 3 seconds after the key is released. If the audio is muted the pressing any of these keys will fade the volume back to its previous level for the device that has the remote focus.
  
Numeric Sets the TV channel, Teletext page or track number of the appropriate application.
  
Rewind This button is application specific. It has the following action dependant on what applicatin has the Remote focus:

Application Action
TV Tuner No action.
Teletext Previous sub-page.
CD Player Rewind current track.
Wave Player Rewind current track.
MIDI Player Rewind current track.
Radio Decrease radio frequency.
  
Forward This button is application specific. It has the following action dependant on what applicatin has the Remote focus:

Application Action
TV Tuner No action.
Teletext Next sub-page.
CD Player Fast forward current track.
Wave Player Fast forward current track.
MIDI Player Fast forward current track.
Radio Increase radio frequency.
  
Play This button is application specific. It has the following action dependant on what application has the Remote focus:

Application Action
TV Tuner No action.
Teletext Show hidden Teletext information (Reveal).
CD Player Play selected track.
Wave Player Play selected track.
MIDI Player Play selected track.
Radio No action.
  
Record This button is application specific. It has the following action dependant on what application has the Remote focus:

Application Action
TV Tuner Save currently paused picture in Device Independant Bitmap format.
Teletext Save current page.
CD Player No action.
Wave Player No action.
MIDI Player No action.
Radio No action.
  
Stop This button is application specific. It has the following action dependant on what application has the Remote focus:

Application Action
TV Tuner No action.
Teletext Change Teletext view Top half, bottom half and normal (Enlarge).
CD Player Stops current CD.
Wave Player Stops current Wave sequence.
MIDI Player Stops current MIDI sequence.
Radio No action.
  
Pause This button is application specific. It has the following action dependant on what application has the Remote focus:

Application Action
TV Tuner freezes currently displayed picture. Pressing this button again un-freezes picture.
Teletext Holds current Teletext page (Hold).
CD Player Pauses CD player.
Wave Player Pauses Wave player.
MIDI Player Pauses MIDI player.
Radio No action.
