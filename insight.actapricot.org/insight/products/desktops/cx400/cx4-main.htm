CX400 (Juneau Motherboard)

Product Code Prefix: KL

 Features
  • Standard 4x4 desktop chassis for ease of servicing
  • Choice of Celeron or Pentium II processors
  • On-board hardware management facilities
    • Motherboard voltage monitoring
    • Temperature monitoring
    • System fan failure
    • Lid removal detection
  • 32MB/64MB SDRAM supplied
  • Choice of 3.2 or 6.4GB Ultra ATA/33 hard disks
  • On-board SoundBlaster compatible Crystal CS4235 stereo sound
  • ATi Rage Pro AGP graphics with 4MB SGRAM
  • On-board networking via Intel 82558 controller
