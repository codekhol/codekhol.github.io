D900		 Included functionality,
Based on IBM SurePath 2.0 code base.
Natoma initial programming as per early BIOS writers guide.
Nat Semi 308 initial programming.
Automatic DIMM detection and sizing.
Pentium Pro cache setup is functional (ignores L2 cache setup options).
Pentium Pro BIOS update is provided for B0, C0, sA0 and sA1 steppings.
Automatic detection, sizing and mode for up to 4 IDE drives.
Single video binary support for GD5462 (Alpha).
Adaptec 7880 binary and setup utility (Beta).
USB controller is initialised.
Signs on with Apricot version string and copyright (+ graphical boot with IBM SurePath logo).

Excluded functionality,
Power management.
Manufacturing support.
Branding.
Setup changes.
Diva card support.
Final chipset optimisation.
Automatic DIMM type, speed and error correction.
Boot block recovery.
Machine ID API for Apricot diagnostics etc.
Cosworth mods’ check.
USB keyboard emulation.
DMI.
Appearance changes.
Pentium Pro BIOS update API.

Known Problems,
Cirrus video controller does not work well in standard VGA graphics mode.
PCI IDE timings have not been optimised.
Some DIMMs are not recognised correctly.
Memory timing is ‘hard wired’.
Nat Semi 308 is showing issues with PnP under Windows 95.
Dual video binary support is not included (GD 5466 or GD 5462).
Pentium Pro BIOS update information is not available in setup.
Processor type and speed determination does not work.

IPR02274, Memory Timing Violation. Chipset Settings Corrected.

IPR02391, IPR02392, Adapter card problems, PCI Bus latency timers should now be correctly configured. These IPR's are no longer reproducible so it is difficult to know whether this is the fix.

IPR02620, CCTEST Failure during MDP, Suspect Drive problems. Not addressed in this release.

FR021, IPR02633, BIOS occasionally reports Configuration Change when there is no change in hardware. Unable to reproduce. Further investigation required.

Some HCT related IPR's were deemed to be problems with the HCT, Details passed to Microsoft. Not addressed in this release.
