CMOS

Clear CMOS - Jumper J10C3

Allows CMOS settings to be reset to default values by jumpering pins 1-2 in J10C3. The system should then be turned off and the jumper returned to pins 2-3 to restore normal operation. This procedure should be done whenever the system BIOS is updated.

To enter CMOS Setup Press F1 during BOOT.
