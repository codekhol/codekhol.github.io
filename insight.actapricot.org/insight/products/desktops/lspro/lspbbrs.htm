BETA		 New drive type 24 added - Maxtor 540 AT.

Support for CE revision of the CL5410 video controller.

Signon and copyright date changed.

Implement various revision made to the XEN-LS II BIOS.
1-04 10/2/93 A fix for systems reporting an invalid floppy drive configuration at power-on. Date and version number updated.
1-05-01
BETA		 17/3/93 Support for Quantum ELS170AT drives has been added.

'Setup' in ROM can now be disabled by using the 'LOCKSET.EXE' program.

The optional ROM is now 'visible at address C000, for remote booting. this fixes a problem introduced with silicon revision 'CE' of the LS Pro.

The date and version strings have been changed.
1-05-02
BETA		 30/3/93 Occasionally, hard disks were not 'seen' by the BIOS when added to a system configured for no drives. This has been fixed.

Drive type 26, Quantum LPS525, is now supported.

A soft reboot used to hang the machine if APRUMB.SYS was being used with anything loaded high. This has been fixed.

Occasionally, hard disks were being 'lost' on power up. This has been fixed.

The version number and date strings have been changed.
1-05-03
BETA		 11/5/93 Machines with Monochrome monitors would hang if security was used. This has now been fixed.

A modification has been made to the Auxiliary device reset logic. This should provide a fix for 'mad' mouse.
1-05 11/5/93 This is a release of 1-05-03 BETA
1-06-01 5/7/93 A number of 'mad' mouse fixes have been incorporated into the keyboard and mouse modules.
1-06-02 5/7/93 A bug which prevented the keyboard from working in Set-up' mode has been fixed.
1-06-04
BETA		 1/3/94 Support for the following drives has been added,

a) Quantum 'Roadrunner' LPS 170 AT
b) Quantum 'Thunderbolt' LPS 270 AT
c) Quantum 'Roadrunner' LPS 340 AT
d) Quantum 'Thunderbolt' LPS 540 AT (Max. capacity: 504MB)
1-06 12/7/94 Chipset programming has been modified to handle the new '3 chip' SIMM's, as in LS Pro Rev 'D'.
1-07-01
BETA		 29/11/94 The latest Ethernet RPL code has been included from LS Pro Rev 'D'.

Rev 'D' Boards
2-00-02
BETA		 30/3/93 Occasionally, hard disks were not 'seen' by the BIOS when added to a system configured for no drives. This has been fixed.

The 'setup' button is ungreyed on graphical boot / the 'setup' message is not displayed on text boot, if 'setup' has been displayed.

The power on password code is not executed if security is enabled

Drive type 26, Quantum LPS525, is now supported.

Occasionally, hard disks were being 'lost' on power up. This has now been fixed.

The version number and date strings have been changed.
2-00-03
BETA		 11/5/93 1.6MB floppy support now works correctly on warm boot.

ROM Setup text for monitors type has been corrected.

POST error messages will now be displayed on graphical boot.

Print Screen is now supported.

A 'mad' mouse fix has been implemented.
2-00-05
BETA		 15/7/93 Various graphical boot 'POST' messages have been altered.

Various 'mad mouse' fixes have been implemented.

If an Apricot boot ROM was fitted then screen corruption would occur on graphical boot. This has been fixed.
