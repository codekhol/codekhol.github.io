LS Pro (Caracal) BIOS Revision Summary (Rev 1.6.4)

Release Date Changes
3-01 11/8/94 ROM SETUP - Floppy access rights, e.g. boot, write or general enable, can now be disabled correctly regardless of the state of system security.
3-02 19/8/94 Security - floppy disable / enable sequence now works without having to power cycle the system.
EEPROM checksum logic has been modified to no longer jump into SETUP.
The 'Disable L2 cache entry is now greyed out if no cache is fitted.
3-03-01
BETA		 30/8/94 Systems should no longer 'hang' on re-boot.
3-03-02
BETA		 1/8/94 The processor speed is now calculated correctly.
The 'Power-on sound' is not played on 'CTL-ALT-DEL.
3-03-06
BETA		 9/9/94 Incorrect PIT values were being programmed if a SMI occurred whilst 'power management' was disabled. This has been resolved.
Cold boot (NOT power-on) caused Windows sound driver to fail initialisation. This has been resolved.
Systems sometimes hung after exiting 'SETUP'. This has been resolved.
Cache is now partially disabled i.e. set to size 0, prior to complete disable. This avoids a system crash, caused by a problem with the chipset.
3-03-08
BETA		 14/9/94 Entering a Power-on password, with graphical boot, would hang the system. This has now been resolved.
3-03-09
BETA		 16/9/94 In 'SETUP' the CPU power management options, although 'greyed out', could be selected. This has been resolved.
'ALT-S' to enter 'SETUP' was being recognised when 'greyed out'. This has been resolved.
The serial and parallel port detection logic has been modified to correctly maintain the BIOS data area (segment 40 hex) contents.
The Microsoft diagnostics program 'MSD.EXE' reported the machine type as being 'AST'. This has been resolved.
3-04-01
BETA		 17/9/94 The cache detection logic has been modified to resolve problems with non-cached machines.
3-05-01
BETA		 26/9/94 The HDU controller can now be disabled by 'SETUP'.
Floppy boot and floppy write can now be disabled by 'SETUP'.
A Microsoft copyright message appeared in memory segment E000. This was from a font file used with graphical boot and has now been removed.
The display is now cleared earlier on reboot.
The ROM checksum has been corrected to prevent the 'JFLASH.EXE' warning message.
The 'ALT-S' for 'SETUP' timing has been altered to match 'greyed' status of 'SETUP' button.
The parallel port can now be disabled by 'SETUP'.
The power-on sound volume 'slider' used to revert to a setting of approximately 95% of full volume after being set at full volume. This has now been fixed.
3-05-02
BETA		 3/10/94 Fixes problem where unable to disable L1 cache.
Fixes problem where Security would hang when text entry was required on a warm-boot.
Support for new FC5 revision of VLSI 82c483. This enables Power Management to be enabled on systems fitted with this revision of silicon.
Cache now set as 9 bit tag, valid bit enable and dirty bit enable (previously 8 bit tag, valid bit disable, dirty bit enable).
3-05-03
BETA		 3/10/94 Fixes problem with configuration of diskless systems.
Fixes problem where system would hang during warm-boot if parallel port was disabled.
Fixes problem where setting Floppy to none and disconnecting floppy cable would cause an Invalid Configuration [Floppy] on every reboot.
3-05-04
BETA		 4/10/94 Fixes problem where if the IDE controller is disabled through SETUP there is a long delay during POST.
3-05-05
BETA		 5/10/94 Fixes problem where Floppy configuration error during POST on every reboot if system configured for Graphical Boot.
3-05-06
BETA		 14/10/94 Systems would hang if the internal processor cache was disabled but the external cache enabled. This has now been resolved.
3-05-07
BETA		 14/10/94 The serial ports would sometimes not be detected on a warm boot. This has been resolved.
Disabling the floppy disk, via 'SETUP', would not prevent the system from trying to boot from the floppy disk, resulting in a disk read error. This has been resolved.
3-05 14/10/94 This release is functionally unchanged from version 3-05-07 BETA.
3-06-01
BETA		 11/11/94 The Norton 'SYSINFO' utility was reporting 'slow seek' or 'seek error' on some hard disk drives (e.g. 'Thunderbolt' & 'Lightning' ). This has now been resolved.
3-06-08
BETA		 21/6/95 The Cirrus Logic video BIOS has been upgraded to version 1.23 to support the rev. 'HH' 543x silicon.
3-06-09
BETA		 9/8/95 PnP 'function 1' was not reporting a failure if invalid parameters were supplied. This has now been resolved.
PnP support for the serial ports only allowed the selectoin of one from two possible base adresses, whereas Windows 95 requires up to four different settings. This has now been resolved.
The PnP 'Set DevNode' function would cause an 'Invalid Opcode' (Exception 6) fault. This was caused by a coding problem and has now been corrected.
PnP 'function 041h' would return an 'Unknown function' code rather than 'Unsupported...'. This has now been resolved.
3-06 17/8/95 This release is functionally unchanged from 3-06-09 BETA.
3-07-01
BETA		 26/2/96 Support for Pentium Overdrive processors has been added.
