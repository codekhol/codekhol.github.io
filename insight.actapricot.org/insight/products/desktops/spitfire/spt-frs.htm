10.06

P/N 161422SW		 17/03/98

 Programming
Target device: Alliance AS29F002T or Micron MT28F002
Utility: AFLASH on update disk (Normally Automated process)
Command line: AFLASH -w -m -f:161422SW.BIN

Changes since release 10.05

  • Signon Changed to 10.06
  • Reboot hanging problem fixed.
  • IPR04230 HD disappearing fix changed from Phoenix fix to Apricot fix.
  • German character support for Branding.
  • Serial / Parallel setup option PnP tracking.
  • Fix for SMART problems (message not being displayed in graphical boot mode, and IPR 4159 - "SMART Warning Message Displays Garbage")
  • Fix for Setup out of sync with Mustang project.
  • Fix for IPR 4251 - Setup message incorrect.
  • Fix for IPR 4466 - Intrusion warning message remains on screen for too short a time.
  • Fix for IPR 4424 - F2 key causes intrusion password entry to not be displayed and machine appears to hang.
  • Fix for IPR 4430 - Pressing F2 has double effect.
  • Fix for IPR 4250 - Int 15h func 87h and 89h not working.
  • Fix for SPR 800439 (Phoenix tracking number) - Screen corruption during Oprom scan.
  • Fix for IPR 4458 - Only one HDD drive powers down during suspend.
  • Fix for IPR 04708 - ATAPI Tape streamer.
  • Suppression of Mitsubishi Logo for OEM boards.
  • Fix for IPR 04693 - Add PME wake on LAN.
  • Fixed AC-Disconnect Security Hole with branding.
  • Default for intrusion clear is now BIOS not OS.
  • Change default CMOS value for cmosLM78Lid to ENABLED from DISABLED.
  • Added fix for IPR 04271 Disable wake-on-ring in Windows 95 does not work.
  • Added fix for IPR 04270 RI Handler that goes to UTR rather than Full On.
  • The message "Press <F1> to..." As will be printed off the bottom of the screen if numerous error messages are found - fixed.
  • Fix GP fault in PCIDIAG.

Known omissions

  • IPR 03816 IO and DMA legacy support.
  • Apricot Diva support
  • ACPI
  • 120M floppy Boot-Block support.
  • German Language support.

Known problems

  • IPR 03625 Floppy drive not ready errors
  • IPR 04017 Power Management does not work under Windows NT4.0
  • IPR 04083 Performance loss with UltraDMA
  • IPR 04193 WFW3.11 will not load without SMARTDrive (Double buffer Issue with PCI bus masters).
  • Fix for UMB space issues / EMM386 hang.
