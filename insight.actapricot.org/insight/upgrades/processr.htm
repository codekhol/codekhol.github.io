Upgrades - MS Series Processors

Introduction

It is very important that before you purchase a processor upgrade for your MS Series computer, you need to make sure that it will be compatible. As a company, Mitsubishi Electric's PC Division only recommend the use of genuine Intel Pentium Processors for MS Series computers.

Before upgrading your computer, make sure that you have identified your computer's Model Number, which is located on the rear of the computer.

When you have identified your computers model number correctly, click on the correct model from the list below for detailed and specific information about upgrading that particular model. At the bottom of each table is extra information (such as BIOS upgrades and processor speed jumper settings) for each machine.

Select Correct Model Number:

  1. MS530 - Model number MKxxxxx-x
  2. MS540 - Model number MTxxxxx-x
  3. MS540 - Model number MMxxxxx-x
  4. MS660 - Model number MFxxxxx-x
  5. MS500 - Model number MWxxxxx-x
  6. MS600 - Model number MVxxxxx-x

Motherboard Upgrade Section
The Trent motherboard that is found in the MS540 (MM) runs the processor at 3.3volts. Pentium MMX processors require the processor to run at 2.8volts.

If a Pentium MMX processor is run at 3.3volts, it will be severely damaged.

SCC (Specialist Computer Centres) can modify a Trent motherboard to run the processor at 2.8volts, allowing you to fit a Pentium MMX processor. Once upgraded, the motherboard will only support Pentium MMX processors.

For more information, contact SCC on 0121-541-4888.

Note: Intel Pentium Overdrive processors have there own voltage regulation onboard and therefore no motherboard modifications are required.
