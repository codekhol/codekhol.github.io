Upgrades - MS Series BIOS

Introduction

All computer hardware has to work with software through an interface. The BIOS gives the computer a little built-in starter kit to run the rest of software from floppy disks (FDD) and hard disks (HDD). The BIOS is responsible for booting the computer by providing a basic set of instructions. It performs all the tasks that need to be done at start-up time: POST (Power-On Self Test, booting an operating system from FDD or HDD).

When upgrading the hardware on your Apricot computer, it may be necessary to upgrade the BIOS as well, for the upgrade to work.

All the current versions of BIOS upgrade for Apricot computers are available below.

Before selecting a file to download, make sure that you have identified your computer correctly, by looking at the MODEL NUMBER on the rear of the computer.

Downloading the file will not in itself upgrade the BIOS. Follow the Installation Instructions below to complete the upgrade.

BIOS Installation Instructions

BIOS Installation Instructions for MS530 (MK) and MS540 (MT)

Once the BIOS update file has been downloaded & saved in your C:\WINDOWS\TEMP folder:

  1. Click on START
  2. Click on RUN
  3. Type in D509COMB.EXE
  4. Press RETURN, then click on UNZIP (this will create three files in your C:\WINDOWS\TEMP folder).
  5. Place a blank formatted disk into the floppy drive A:
  6. Click on START
  7. Click on RUN
  8. Type C:\WINDOWS\TEMP\IMAGE 509BIOS.IMG A: /M /B
  9. When the boot disk has been created, restart the computer and boot from the diskette.
  10. From the menu press 1 to detect the type of video controller, and then press 2 or 3 to upgrade the BIOS for your type of video controller.

BIOS Installation Instructions for MS540 (MM) and MS660 (MF)

Once the BIOS update file has been downloaded & saved in your C:\WINDOWS\TEMP folder.

  1. Insert a blank formatted diskette into the floppy drive
  2. Click on START
  3. Click on RUN
  4. Type in C:\WINDOWS\TEMP\TRENT708.EXE A:
  5. When the boot disk has been created, restart the computer and boot from the diskette.
  6. When the BIOS update has completed, remove the diskette and restart the computer.

BIOS Installation Instructions for MS500 (MW)

Once the BIOS update file has been downloaded & saved in your C:\WINDOWS\TEMP folder:

  1. Click on START
  2. Click on RUN
  3. Type in C:\WINDOWS\TEMP\MSTNG_15.EXE (this will create three files in your C:\WINDOWS\TEMP folder).
  4. Insert a blank formatted diskette into the floppy drive
  5. Click on START
  6. Click on RUN
  7. Type in C:\WINDOWS\TEMP\IMAGE MUSTANG.IMG A: /M /B /Q
  8. When the boot disk has been created, restart the computer and boot from the diskette.
  9. When the BIOS update has completed, remove the diskette and restart the computer.
