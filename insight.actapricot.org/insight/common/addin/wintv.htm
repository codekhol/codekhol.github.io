Win/TV Prism Card (Rev 1.9.1)

Jumpers and Connectors

I/O Address Jumper JP1

The Win/TV-Prism card can be set for one of 8 different I/O locations, between 500 - 5E0. The default Win/TV address is 500. To configure the I/O address, use the 6 pin jumper labeled JP1 in the upper left corner of the card.

I/O Address INI File Pins 1, 2 Pins 3, 4 Pins 5, 6
500 - 506 Port = 500 ON ON ON - Default
520 - 526 Port = 520 OFF ON ON
540 - 546 Port = 540 ON OFF ON
560 - 566 Port = 560 OFF OFF ON
580 - 586 Port = 580 ON ON OFF
5A0 - 5A6 Port = 5A0 OFF ON OFF
5C0 - 5C6 Port = 5C0 ON OFF OFF
5E0 - 5E6 Port = 5E0 OFF OFF OFF

After changing the I/O address, the I/O port address in the PCVIDEO.INI file (found in the C:\WINDOWS directory) needs to be changed. Look in PCVIDEO.INI for a line that says:
Port=500
Change this line for the I/O address selected. See the chart above for the INI file line

IRQ Jumper JP2

An IRQ is not required to operate Win/TV, but programs like Video for Windows need an interrupt for capturing full motion video. The Default IRQ position is 15. If IRQ conflicts occur, change jumper selection or disable the IRQ by removing the jumper.

Other Jumpers & Connectors

J1 VGA Loop Cable connector 8 pin round DIN
J2 Audio line level output 1/8" stereo/mono jack
J3 VGA Monitor connector DB15 pin
J4 Audio/Video Input connector 4 pin round DIN
P5, J5 Video option connectors Used for optional hyperCOM teletext security device for the Reuters 1000 News Service
P7 Internal Line out audio to sound card; not normally used
P6 Feature Connector internal cable to VGA card
