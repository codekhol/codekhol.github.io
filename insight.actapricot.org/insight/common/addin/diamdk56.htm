Diamond K56 modem

Features

  • Speeds up to 56,000 bps on data transfer
  • 14,400 bps (ITU V.17) fax capable
  • V.80 (video conferencing) capable
  • Full Duplex Speakerphone with Echo Cancellation
  • Single phone-line access to voice mail, fax, Internet, and file transfer

Data details

  • Speeds: 56,000 / 54,000 / 52,000 / 50,000 / 48,000 / 46,000 / 44,000 / 42,000 / 40,000 / 38,000 / 36,000 / 34,000 / 33,600 / 32,000 / 31,200 / 28,800 / 26,400 /24,000 / 21,600 / 19,200 / 16,800 / 14,400 / 12,000 / 9600 / 7200 / 4800 / 2400 / 1200 / 300 bps
  • Protocols: Rockwell K56 / ITU-TSS V.34 / ITU-TSS V.32bis V.32 / V.23 / V.22bis / V.22 Bell 103 / 212A and V.21
  • Error Correction and Data Compression: ITU-TSS V.42bis and MNP 2-5
  • Auto answer / auto dial
  • Industry-standard and extended AT Command set
  • V.80 synchronous access mode
  • Improved download speeds available when connecting to compatible 56Kbps server

Fax details

  • Speeds: 14,400 / 12,000 / 9600 / 7200 / 4800 / 2400 bps
  • Protocols: ITU-TSS V.17, V.21, V.27ter, V.29
  • Compatible with Class 1 fax software. Works with Group 3 fax machines
