3Com Ethernet 3C509 ISA Card (Triple Combo) (Rev 1.8.1)

Specification

EtherDisk Software

EtherLink III adapters ship with a 3Com EtherDisk that contains the drivers for leading network operating systems:

  • Novell NetWare versions 2.x, 3.x, and 4.0x [client (OS/2, DOS) and server]. Personal NetWare version 1.x.
    NDIS 2.01: both DOS and OS/2 drivers; supports MS® LAN Manager, Banyan® VINES® (client), LANServer, DEC® PATHWORKS, (TM) and Lantastic. NDIS 3.0 for Windows NT and Windows for Workgroups.

    • Other Software

    Drivers for other third party software such as SCO, UnixWare, (TM) Solaris and Interactive are available from 3Com's BBS system or the third party.

    BNC Connector

    For use with on-board transceiver; supports network segments up to 1000ft / 305m with thin Ethernet coaxial cable (RG58 C/U or A/U).

    RJ-45 Connector

    For use with on-board transceiver; supports Level 3 unshielded twisted-pair network segments up to 328ft / 100m.

    AUI Connector with Slide Lock

    For attaching a transceiver drop cable and IEEE 802.3-compatible external transceiver;
    supports network segments up to 3280ft / 1000m with thick Ethernet cable.

    Physical Dimension

    Length: 3.95 in / 10.0 cm
    Width: 6.14 in / 15.6 cm

    Power Requirements: Operating Voltage

    +5V ±5% @ 150mA max.
    +12V ±5% @ 500mA max.

    Environmental Operating Ranges

    Temperature: 32 to 158 °F (0 to 70 °C).
    Humidity: 10% to 90% non-condensing.
    Altitude: to 9800ft / 3000m.

    Safety and Electromagnetic Compatibility

    EN55022, FCC Part 15 Subpart J Class B, VDE 0871, Class B, VCCI.
