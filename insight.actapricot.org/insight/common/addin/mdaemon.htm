MDaemon E-mail Server 

What is MDaemon?

MDaemon is a SMTP/POP3 mail server software package for Windows 95 and NT. This is typically the type of mail server software required to send e-mail to users on a LAN and also to the Internet. MDaemon allows an organization to implement a inexpensive but powerful mail system that will allow users to e-mail one another, and users on the Internet without the need for expensive gateways, routers, or dedicated lines.

What’s so unique about MDaemon is the ability to route mail to an entire network through one connection and mailbox from an Internet Service Provider (ISP). If a company has just one dial-up access account from a provider with a single mailbox they can provide e-mail service to their entire organization.

How does it do that? ISP’s have the ability to forward all mail headed to a destination to a single e-mail box. So, let’s say for example that you have a mailbox at your ISP called: someone@somewhere.com. This is called a POP3 (Post Office Protocol Version 3) account. Your ISP can send anything headed to somewhere.com to that single mailbox, for example john@somewhere.com, louise@somewhere.com, bill@somewhere.com, and so on and so forth. MDaemon can log into this single mail box and retrieve this mail, sort it out and send it to the matching mailbox that you’ve defined in MDaemon. Not only that, but it can do it at any interval that you define from one minute on up. It will also dial your account automatically to check for and send new mail.

If you have more than one POP3 account you want to check, MDaemon has what’s called MultiPOP. You can check as many POP3 accounts as you like, however only one of the POP3 accounts can be parsed for aliased mail. The other POP3 accounts mail would be delivered to whatever single mailbox that you define in MDaemon for that particular POP3 account.

MDaemon supports several types of connections to the Internet including modem, ISDN, leased, etc. MDaemon is scaleable, so it can grow as your organization's mail needs grow.

MDaemon supports listserv, aliasing, auto responders, auto forwarding, multiple domains, remote administration, and lots of other features. The unlimited version supports 5000 users, so you can use MDaemon anywhere - from the smallest LAN to a large organization.

You can use a single dial-up account to not only provide Internet and internal mail to your LAN, but general Internet access as well. MDaemon is fully compatible with Qbik’s WinGate, and will use the same connection that WinGate is using to provide simultaneous WWW access (and other applications ) to the Internet.

MDaemon works with any Winsock compliant POP3 mail software such as MS Exchange, MS Internet mail, Eudora, Pegasus, Netscape, E-mail Connection, and many others.
