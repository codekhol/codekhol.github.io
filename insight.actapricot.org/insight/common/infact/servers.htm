System Overview

With the fault-tolerant Apricot FT//e and the FT//s SuperServer, Apricot is one of the first manufacturers to realise the full potential of industry standard processing for large-scale host and network server applications.

Apricot fault-tolerant servers couple the power of the Intel 486 and Pentium™ processor families with dedicated host architecture - including high performance disk arrays, fault-tolerant features, security and large-scale expansion capability - providing a hardware platform with the ability to replace many mini and mainframe computers.


Technical Highlights

Departmental Applications

Both the FT//e and FT//s are built around the same motherboard and core functionality. The FT//e provides a highly cost-effective workgroup solution for small-medium LANs of ten users upwards, whilst the massive expansion of the FT//s SuperServer provides the extra capacity needed for major enterprise-wide applications.

Apricot server development has been carried out in close co-operation with the main operating system developers. Apricot is one of the few manufacturers in the world to have a Novell engineer dedicated to its products at Novell's headquarters, as part of our Strategic Engineering Support (SES) agreement.

Apricot also has a very close working relationship with Microsoft, encompassing a strategic TAM (Technical Account Management) agreement, which guarantees full support for Apricot products. In addition, Apricot is also a major OEM for SCO.

These relationships mean that every system is optimised for performance with the main network and multi-user systems, including Novell NetWare, UNIX, LAN Manager and Windows NT.

Individually Configured

Every Apricot server is individually configured and assembled to meet the customer's exact specification.

Options include up to 192MB of on-board RAM (256MB on Pentium™ systems), and powerful drive arrays giving up to 20GB of storage capacity on the FT//s SuperServer.

All Apricot server drive systems have been developed for exceptional throughput, using Apricot's ultra high-performance Multiple Drive Array (MDA) controller.

SuperServers

Unlike conventional PC servers, Apricot servers provide a significant level of fault-tolerant protection as standard.

Every system unit includes an uninterruptible power supply and built-in temperature sensors. And with options for drive mirroring, drive duplexing, RAID and full server mirroring, Apricot server technology provides the highest possible level of protection to ensure continued system availability.

In addition, the Apricot LOC Technology security system provides the physical protection that is essential to guard unattended servers against tampering and unauthorised access.

Apricot Multiple Drive Array

Apricot MDA controller

High bandwidth, multi-threaded SCSI II controller ensures fastest possible throughput of data between disk array and CPU. Due to its low command overhead, integral processor and optimised drivers, data can be routed through the controller in just 300 microseconds (up to 3 times faster than most controllers).

Quad controller option (dual on FT//e)

Provides increased performance and first class fault tolerance.

Supports Fast SCSI II

With data transfer rate of 10MB/sec, provides double the throughput of conventional SCSI - matching the outstanding performance of Apricot drives with high-speed throughput.

Streaming Data Protocol

Apricot MDA supports Micro Channel II 40MB/sec data transfer - increasing throughput between SCSI interface and CPU.

Drive mirroring and duplexing under NetWare, Windows NT, LAN Manager and Univel UnixWare

Provides maximum protection for system availability against drive system failure. Mirroring duplicates entire disk system, to provide a continuous backup: duplexing duplicates both disk system and controller, for additional protection.

Supports data striping under NetWare and Windows NT

Data interleaved across an array of drives for increased performance on multiple read/write operations.

Multiple drive arrays

Supports arrays of up to ten drives, giving a potential maximum capacity of 20GB on FT//s (6GB on FT//e) - hence providing the expansion needed for major database applications.

Ultra high performance drives

Fastest drives currently available within the 3.5" form factor - combining very fast rapid access with exceptional reliability.

Apricot Technology

Multiple Drive Array (MDA)

Apricot MDA is a high bandwidth, multi-threaded drive array controller that is designed to transfer data to and from the CPU at very high speeds. To achieve this level of throughput, the MDA approach exploits the fastest available technology at every stage of the data transfer path - including high performance drives, fast SCSI II interface and Micro Channel II bus.

The MDA controller has been designed to reduce the overhead involved in transferring data from disk to processor to an absolute minimum. It has a built-in 16-bit processor to streamline the handling of SCSI commands, and it's supported by a superbly efficient range of software drivers.

The result is an unparalleled level of raw throughput, typically with just 300 microseconds taken to route data from the SCSI bus to the Micro Channel bus.

Conventional RAID 3, 4 and 5 systems use intelligent array controllers to protect against failure with complex redundant data systems. Apricot MDA uses RAID level 1 disk mirroring or duplexing to ensure continuous system availability, using the proven features of operating systems like Novell 3.11. With no need for complex parity checking routines, this approach provides faster performance than conventional RAID 3, 4 and 5 subsystems.

Fault Tolerance

In any major host or server system, protection and control is as important as performance and capacity. Apricot servers are the first industry standard systems to provide effective safeguards against component failure, power supply problems and unauthorised access.

The basic concept behind this strategy is 'the machine room in a box'. As far as possible, Apricot servers recreates the control and protection of the traditional computer room inside the server system unit.

All Apricot servers include an integral UPS (uninterruptible power supply) unit. If there's any break in the mains supply, the UPS takes over automatically. Even outside the machine room, the server is still protected by a level of environmental control, with built-in temperature sensors.

The system also monitors the status of its own UPS, checking that the battery is fully charged and ready for operation.

Apricot's server fault tolerance is integrated with the main network operating systems, providing automatic messaging and remote monitoring facilities throughout the network.

Apricot LOC Technology

The Apricot LOC Technology security system is the second key element of its server protection strategy. With access controlled by both a password and an infra-red transmitter, it provides the physical protection that is needed to guard unattended servers against tampering.

On its own, a password is inherently insecure. It leaves the system at the mercy of careless users and skilled hackers. Using the Apricot approach, access is controlled by special hardware integrated into the motherboard - with a separate security processor, battery power and memory, this means that intruders can't hack into password data from the main operating system.

Without the right combination of password and infra-red signal, it's impossible to boot an Apricot server. In addition, the LOC Technology barrier provides tamper-proof protection while the server is running - protecting removable media and barring access to the inside of the system unit.

64-bit Motherboard

The Apricot fault-tolerant servers are designed with a 64-bit architecture in mind to provide first class performance. With a choice of 486 or Pentium™ processors, the Apricot SuperServer is at the forefront of technology, providing the highest levels of performance at an affordable price.
