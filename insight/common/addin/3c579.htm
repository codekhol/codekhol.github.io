3Com Ethernet 3C579 EISA Card (Rev 1.9.1)

Specification

EtherDisk Software

EtherLink III adapters ship with a 3Com EtherDisk that contains the drivers for leading network operating systems:

  • NetWare 4.x & 3.x server.
    DOS ODI.
    OS/2® ODI.
    NDIS 2.01 DOS.
    OS/2.
    Windows for Workgroups
    Windows NT
    •

    Other Software Drivers for other third party software such as SCO, UnixWare, (TM) Solaris and Interactive are available from 3Com's BBS system or the third party.

    BNC Connector For use with on-board transceiver; supports network segments up to 1000ft / 305m with thin Ethernet coaxial cable (RG58 C/U or A/U).

    RJ-45 Connector

    For use with on-board transceiver; supports Level 3 unshielded twisted-pair network segments up to 328ft / 100m.

    AUI Connector with Slide Lock

    For attaching a transceiver drop cable and IEEE 802.3-compatible external transceiver; supports network segments up to 3280ft / 1000m with thick Ethernet cable.

    Interrupt Levels

    3, 5, 7, 9, 10, 11, 12, 15.

    I/O Base Address 16 possible addresses, 200 to 3E0.

    Tri ROM (formerly Boot ROM)

    Updateable Tri ROM for NetWare, RPL and BootP (TCP/IP) .

    ROM Socket Memory Address 8, 16, and 32-KB windows supported from C0000 to DE000.

    Physical Dimension

    Length: 4.1 in / 10.4 cm
    Width: 7.3 in / 18.6 cm

    Power Requirements: Operating Voltage +5V ±5% @ 1.5A max.
    +12V ±5% @ 0.5A max.

    Environmental Operating Ranges

    Temperature: 32º to 158 °F (0 to 70 °C).
    Humidity: 10% to 90% non-condensing.
    Altitude: to 9800ft / 3000m.

    Safety and Electromagnetic Compatibility FCC Part 15 Class B; VDE 0871 Class B when used with shielded twisted-pair wiring; UL1950, 1st edition..
