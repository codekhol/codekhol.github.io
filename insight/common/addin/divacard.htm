Apricot LS Security Card (Diva) Specifications (Rev 1.11.1)

 

LS Security card (Diva) binary, PN 157278SW

Introduction
This is the initial beta release of the LS Security card ROM binary. This controls operation of the card during BIOS POST. The software is based upon the code originally used for the 'Feature' card product.
