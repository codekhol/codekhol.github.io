DPT SmartRAID IV Ultra SCSI RAID Product (Rev 1.13)

PM3334UW

Key Features

•On-board high performance processor optimized for handling complex RAID algorithms
•SCSI, SCSI-2, SCSI-3 and Ultra Wide SCSI support with active termination
•Supports up to 40MB/s Ultra Wide synchronous SCSI transfers
•Overlapped I/O and tagged command queuing for increased SCSI performance·
•Compatible with all major operating systems
•Four on-board SIMM sockets support for up to 64MB of hardware cache
•Support for hardware RAID 0, 1 and 5
•Equipped with flash ROM for easy firmware upgrades·
•Optional error correcting memory and data path protection
•Includes Storage Manager software for easy installation, RAID setup and maintenance, advanced storage diagnostics and performance tuning

Technical Specifications

Host interface: 16-bit ISA, 32-bit EISA
On-board CPU: 68040, 40Mhz
Max Host Transfer Rate: 33MB/s (EISA), 132MB/s (PCI)
SCSI Interface: SCSI, SCSI-2, SCSI-3, Fast, Fast Wide, Fast-20 Wide Scatter/Gather, Command Queuing, Disconnect/Reconnect
Max SCSI Transfer Rate: Up to 40MB/second Fast-20 Wide Synchronous
SCSI Bus: 16-bit single ended, active termination 16-bit differential (PM3334UDW)
Internal Port: 68-pin high density
External Port: 68-pin high density

DC Power Requirements

Voltage: 5V +/- 5%
Ripple and Noise: 50mV peak-to-peak max
Current: 2.20A typical (PCI), 2.40A typical (EISA)

Environmental Specifications (Operating)

Ambient Temperature: 0 to 50 C
Relative Humidity: 10% to 90% non-condensing
Altitude: 3,000 meters (10,000)

RAID Support*: 0, 1 and 5
