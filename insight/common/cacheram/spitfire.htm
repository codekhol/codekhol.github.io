Memory Configurations

DIMM specification

Memory is fitted in two vertical DIMM sockets which must be populated with 3.3V, 64-bit unbuffered 4-Clock SDRAM type DIMMs.

The two DIMM slots accept DIMMs of 16, 32, 64 and 128 MBytes in any combination, to the maximum of 256 MBytes.

Note that these DIMMs are of the Synchronous DRAM type (SDRAM) and are not the same as those used with Cosworth, Discovery and PCL6000 motherboards. They are the same as those used with the Lightning and Mustang motherboards.
