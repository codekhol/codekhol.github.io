This machine is available as part of the Apricot Fastbuild service and, as such, there are various memory options available.

486 Systems have a maximum available RAM of 192MB
Pentium™ Systems have a maximum available RAM of 256MB

 

Memory Configurations - 486 Based Processor Board

192MB Maximum.
Six 36 bit SIMM sockets.
Minimum system configuration = 16MB.
32MB SIMMs are industry standard and are not the same as those used in the XEN-LS II / XEN-PC.
Memory is configured as 64 bit wide, thus SIMMs must be installed in identical pairs, e.g. 16MB = 2 x 8MB SIMMs.

Memory Configurations - Pentium Based Processor Board

256MB Maximum.
Eight 36 bit SIMM sockets.
Minimum system configuration = 16MB.
32MB SIMMs are industry standard and are not the same as those used in the XEN-LS II / XEN-PC.

Memory is configured as 128 bit wide, thus SIMMs must be installed in two groups of four, as follows:

MM1, MM3, MM5, MM7
MM2, MM4, MM6, MM8

e.g. 16MB = 4 x 4MB SIMMs
