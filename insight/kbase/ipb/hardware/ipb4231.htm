Installing Eicon Diva 2.0 ISA ISDN card into an FT2400 running NT 4.0

Prior to beginning the ISDN card installation, it would be worthwhile copying the !GDI000A.CFG file from the WIN_NT\ENGLISH\DISKS\I386\DRIVERS1 folder on the Eicon install CD to the SCU floppy. Also, make sure that the NT4.0 service pack 3 has been installed.
Remove the left-hand side panel of the FT2400 and insert the card into one of the ISA slots. At this point of the installation, standard anti-static precautions must be observed.
When the card is fitted, replace the side panel and boot the FT2400 using the SCU Floppy Disk. This can be created from the System Utility CD that is supplied with the server. When the SCU utility is started, enter Step 2: Add and Remove Boards. This will then show a list of devices found in the installed in the server. It may be necessary to add the ISDN card to the list, although during testing it was auto-detected. If you have to manually add the card, then select the EICON.DIEHL DIVA 2.0 ADAPTER.
As default this is allocated to IRQ5 and I/O range 200-207 by the SCU. Leave on IRQ5, but change the I/O range to 240-247. Once that is done, exit back to the SCU main menu, and save the configuration. Once the configuration is saved, exit the SCU utility, and restart the machine.

Log into NT as Administrator or equivalent, insert the ISDN Software Suite CD into the CD drive, and from the root of the CD, run the CDSETUP file.
The start-up screen will appear giving various start-up options. Select the Windows NT English option, and when the second screen appears, select Start Diva for Windows NT Installation now
The next windows box to appear is for selecting the actual card you have installed. In this case the card is Eicon.Diehl DIVA 2.0 Adapter. Select and click OK. This will then start installing some drivers.
The next dialogue box is to let you know that the server contains more than one hardware bus and to let you know where the server has detected the ISDN card. Typically, the type will be ISA, and Bus Number 0. Accept the values presented by clicking OK.

At the next dialogue box, titled EICON.DIEHL DIVA 2.0 Adapter Set-up (2.01) there are some changes to be made. The FT2400 SCU assigns IRQ5 to the ISDN card, whereas the ISDN card’s own software assigns IRQ10. Change this to 5. It will correctly detect the I/O address as starting at 240.
The third option – titled ISDN Protocol (Switch Type) will be blank. Clicking the reveal arrow will reveal the various ISDN settings for different countries. Select Europe and other countries, Euro_ISDN (E-DSS1), leave the rest of the settings as they are, and click the OK button.
At the next dialogue box, accept the default button click. When the Port List appears, click OK, and accept the default button click at the next dialogue box. At the next dialogue box asking you to enter the path to the NT install files, insert your NT Server CD, and type the path as D:\I386 (assuming your CD drive is drive D:\). This will then load the Windows NT Remote Access Service (RAS). The RAS Capable Device will be shown as ISDN1 – DIWAN, this is correct, click OK, and the dialogue box will disappear leaving the Remote Access Set-up showing the port, device and type. These are all correct, but there are some other options that need changing at this point.
Click the Configure button to give you the usage types. As default it only selects Receive Calls Only. We recommend setting this to Dial Out and Receive Calls. Click OK to close that dialogue box, and then click the Network button.
This option is for network protocols required to connect to the server. To make it as easy as possible, de-select the IPX tick boxes, but make sure that the TCP/IP tick boxes are selected. All other options and settings are correct, so click on the OK button to return to the Remote Access Set-up screen. At this screen, click continue. At the next dialogue box telling you that RAS has been successfully installed, click OK, and select the default button of the next dialogue box. When the WAN-Miniport Adapter Set-up (2.01) appears, you will need to remove your NT Server CD, and replace it in the CD Drive with the ISDN Software Suite CD. Once this is done, and the CD activity light has gone out, click the Continue button. At the next dialogue box, click OK, and when the Controller List box appears, this should display 1 [2] "My DIVA 2.0 ( 2)" Eicon.Diehl DIVA 2.0 Adapter.
Click OK to continue loading software, and at the next dialogue box, click OK. The NT Network properties window will appear click OK to close it, and when you are asked to restart your computer, click OK.
The FT2400 will then re-boot, and you will then be able to use the ISDN facility.
