Configuring Third Party IDE Drives in Apricot Machines

This IPB should be read in conjunction with IPB 4143.

When installing third party Hard disk drives into Apricot machines the following points should be noted.

Samurai based Machines SA/SD/SF XEN-LS II, XEN-PC

These machines only support user defined drive types in the later BIOS's (1.06 for rev D & 2.05 for rev E/F) and above.

In setup select the user defined option, next select the UserHDs button. Next select the Detect1 button. If the drive is correctly configured it will display its parameters. Save and exit from setup.

The drive should boot or enable an operating system to be installed.

If the machine brings up the setup screen again enter the user defined section as above and reduce the cylinder count by 1, leaving the head and track count the same. Save and exit from setup.

This time the drive may function correctly, if not repeat the above process reducing the cylinder count again. It has been found that by reducing the cylinder count by 4 or 5 the drive will be recognised.

A1 & J3 based Machines SH/SJ XEN-PC, XEN-PCm, XEN-PCLS
VA EPx

Enter setup & select type 255 for the first fixed disk. Now press the F8 key, this will interrogate the drive and set the parameters for the drive.

Jade/Ruby & Revenge RA/RB/RC XEN-PC, XEN-PCm, XEN-PCLS
& P2 based machines RD/RE/RF/RG/RH

These motherboards incorporate a hard disk drive autodetect function.

Bonsai based Machines BO/BN LS Pro 386SX-33 & 486SLC-33

There is no support for third party drives in the Bonsai based LS Pro.
