FTs Switch Settings

A small number of instances have been reported, where motherboard switch positions have been incorrectly set at the manufacturing stage. Not all switch functions are detailed on the system legend and may therefore go un-noticed by installers. The most common fault is the incorrect setting of SW5 which informs the BIOS which type of machine the motherboard is installed into (Qi 486 or FTs). The result of this is that if UPS monitoring software is loaded, the system may not complete its BOOT sequence correctly.
