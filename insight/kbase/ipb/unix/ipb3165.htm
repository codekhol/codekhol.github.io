Recent SCO Supplements applicable to Apricot
UOD374 New driver for High Sierra filesystem to let binaries run when resident on CD-ROM. For use on Open Desktop 2.0/NFS 1.2.0 with NET362B or Open Server 3.0/NFS 1.2.1 with NET373.
UOD376 SCSI tape supplement to handle variable block sizes on QIC-525 tape units and other units of greater than 150MB which which support this facility. For use on UNIX 3.2v4.[12], Open Desktop 2.0 or Open Server 3.0.
UOD378 Fixes problem with AIO driver and memory allocation (malloc error). The AIO driver is not by default used for disk I/O but any database vendor using raw disk I/O may require this fix if mkdev aio has been run. For use with Open Desktop 2.0/Open Server 3.0/UNIX 3.2v4.[12].
UOD381 Fixes problem with AIO driver and growing a structure in shared memory. As above the AIO driver is unlikely to be used by customers not using raw disk I/O. For use with Open Server 3.0/UNIX 3.2v4.2.
NET382 Supplement correcting problems with permissions using rcp with wild cards and a number of rare panic conditions with tcp and netbios. For use with Open Server 3.0 or systems with TCP/IP 1.2.1 installed. We strongly recommend this supplement is installed.
