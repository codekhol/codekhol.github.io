SCO Supplements for routine installation

Below is a list of supplements that should routinely be applied to SCO Unix 3.2v4.2 / TCP 1.2.1 / NFS 1.2.1 / Open Server 3.0. Reference should also be made to IPB 3165 and IPB 3154 which also refer to some of the following and routine supplements for previous versions of Unix 3.2v4 / TCP 1.2 / NFS 1.2.

Supplement Suitable for Details
UOD368B Unix 3.2v2
Unix 3.2v4.[012] 		Replacement /bin/passwd. Fixes the -l and -u option removing the users account.
UOD376A Unix 3.2v4.[012] Replacement SCSI tape driver. Has cured various SCSI errors during backups.
UOD369B Unix 3.2v4.[012] Console keyboard supplement. Cures "sticking" Alt (& other) keys.
UOD374A Unix 3.2v4.[012] CD-ROM executable supplement. Cures "killed" error message when trying to run executable file from CD and trailing dot on filename.
NET373A NFS 1.2.1 Mounting CD-ROM over NFS
NET382D TCP 1.2.1 Enhancement Supplement. Fixes non passing of TERM variable with rlogin, intermittent corruption during rlogin. Streams buffer allocation fixes, PPP fixes and others. Supersedes NET382C
LLI 3.4
(EFS 140) 		TCP 1.2.1 Latest SCO supplied LLI drivers. Refer to IPB 3175
NET387A TCP 1.2.1 Updated AMD Ethernet driver. LLI 3.4 must already be installed.
UOD378A Unix 3.2v4.[012] Cures malloc failure with errno 16 (EBUSY) under heavy load with the AIO driver. Only needed if AIO driver has been configured with mkdev aio - aioinfo command will fail if AIO driver is not configured.
UOD381A Unix 3.2v4.2 Cures malloc failure with errno 16 (EBUSY) and growing a structure in shared memory. As with UOD378A above, the AIO driver is unlikely to be used unless raw disk I/O is needed - use aioinfo to verify.

These disc images and documentation files are available from Apricot Insight BBS, our Internet server, our UUCP Unix server, and the Insight CD rel. 6. Refer to IPB 3178 for further information regarding these locations.
