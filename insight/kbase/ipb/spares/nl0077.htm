SHOGUN Upgrades

The following is a guide to upgrades for SHOGUN (Model Prefix: HA) to include Processor, Hard Disk, Memory & options/implications on upgrading to FT4200 specification for each :-

Option 2 - Upgrade to Pentium Pro ‘Discovery’ motherboard :-

Upgrade to Prod Code RRP CAT
Single Pentium Pro 200 HA008 £2500 A
Dual Pentium Pro 200 HA009 £4000 A
Quad Pentium Pro 200 HA010 £7500 A

Notes :-
(i) All existing SHOGUN memory SIMM’s must be replaced with FT4200 DIMM’s (see Memory Upgrades - Option 2 below).
(ii) Requires REV.B Shogun Power Dist PCB.
(iii) ‘Discovery’ motherboard has 1 x 7880 (HD) & 1 x 7850 (DDS/CD) on-board controller & therefore requires controller card for each additional SCSI channel on non-RAID systems.

Option 2 – Upgrade to FT4200 SCA2 ‘Ultra-Wide’ HDD’s :-

Item PT / Prod Code RRP CAT
4GB SCA2 ‘Ultra-Wide’ HDD PT15756331 £529 A
9GB SCA2 ‘Ultra-Wide’ HDD PT15756431 £699 A
SCA2 HDD Module (4 bays) HB58486 £200 A
DPT PM3334UW 3-CH RAID HB58024 £1399 A
DPT RAID Cable Kit HB57832 £100 A

Notes :-
(i) SCA2 HDD Modules (HB58486) must be fitted when using SCA2 ‘Ultra-Wide’ HDD’s.
(ii) SCA2 HDD Modules will also support SCA1 ‘Wide’ HDD’s.
(iii) Cannot mix SCA1 ‘Wide’ & SCA2 ‘Ultra-Wide’ HDD’s on a single UW RAID controller.
(iv) 50-WAY SCSI CABLE PT15552431 & 68-WAY to 50-WAY CONVERTER PT15552530 required for DDS/CD support if upgrading to SCA2 HDD Modules on existing SHOGUN motherboard.

(i) Memory card supports 12 SIMM’s (3 banks of 4) up to maximum 384MB.
(ii) Second memory (PT15487131) card must be added to fully populate up to maximum SHOGUN configuration of 768MB.

(Only available on SHOGUN upgraded to Pentium Pro ‘Discovery’ motherboard as per Section 1 - Option 2).

(i) ‘Discovery’ motherboard 8 DIMM’s (2 banks of 4) up to maximum 2GB.

iii) All DIMM’s in a bank must be the same.
