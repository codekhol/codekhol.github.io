Installation of NetWare Requester for OS/2 2.0

Following recent installation of NetWare Requester for OS/2 V2.0 in the Network Support Group, the following installation guidelines have been recommended.

Requirements

Software:-

OS2MAN.ZIP - Requester Manual for OS/2 V2.0
REQSTR.ZIP - NetWare Requester for OS/2 V2.0 (Disk 1 - Install)
OS2UT1.ZIP - NetWare Requester for OS/2 V2.0 (Disk 2)
OS2UT2.ZIP - NetWare Requester for OS/2 V2.0 (Disk 3)
NSD201.ZIP - Update to NetWare Requester for OS/2 V2.0
82596ODI.SYS - ODI driver for Ethernet 596 card.
NET.CFG - NetWare configuration file

Download the above files from AREA37 on the bulletin board and unpack them by using PKUNZIP filename -D (To unpack the directory structure).

Note: The NetWare Requester Install disk must be labelled as REQUESTER. The installation procedure will not complete if the disk has an incorrect label.

The files 82596ODI.SYS and NET.CFG are also available on the Apricot 596 Network Drivers Disk 1 Rel 2.2 in the following sub-directory: \NETWORK\ODI\OS2.

Installation Procedures

Open an OS/2 Window. Insert the NetWare Requester Install disk into drive A and type A:INSTALL and press RETURN.

From the NetWare Requester for OS/2 Installation Utility menu choose:-
Installation
Requester on Workstation

From the Set Target Directory screen accept the default target directory for the Requester files:
C:\NETWARE and the default Source drive as A. Select OK.
From the Requester Installation screen accept Edit CONFIG.SYS and Copy Files. Select OK

From the Select Options for CONFIG.SYS screen, Choose the required Network Interface Card
Type the ODI driver - 82596ODI.SYS.

Select SPX Support for OS/2 Sessions and Netbios Emulation for OS/2 Sessions. Select Save.
Select YES to 82596ODI.SYS being a third party NIC driver.

Type the path and filename where the network driver is located. e.g. A:\82596ODI.SYS. Insert the 82596ODI network driver disk into drive A.
Select OK.

From the Save Changes to CONFIG.SYS screen, Save file as C:\CONFIG.SYS. Select OK.
From the Copy Requester files screen accept Requester files will copy to C:\NETWARE. Select Copy.

Select YES to install NetWare Tools and Remote Printer Utilities in this desktop folder.
Select OK when all files have been copied. Exit the install utility.

Copy the NET.CFG file from the network drivers disk to C:\NET.CFG.
Edit the CONFIG.SYS file. Add the line
DEVICE=C:\NETWARE\ODINSUP.SYS before the
DEVICE=C:\NETWARE\82596ODI.SYS line and save the file.

Shutdown the OS/2 workstation and re-boot.

Applying the NSD update to NetWare Requester

Open an OS/2 Window. Insert the Update to NetWare Requester disk into drive A and type A:INSTALL and press RETURN

From the NetWare Requester for OS/2 Installation Utility menu choose:-
Installation
NSD on Workstation

From the Set Target Directory screen accept the default target directory for the Requester files:
C:\NETWARE and the default Source drive as A. Select OK.

From the Copy Requester files screen accept Requester files will copy to C:\NETWARE. Select Copy.
From the NIC Drivers to Copy screen select Copy only the default driver and select OK.
Select OK when all files have been copied. Exit the install utility.

Shutdown the OS/2 workstation and re-boot.
