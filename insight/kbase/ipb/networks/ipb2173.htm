Greater than 16MB of RAM in FT//ex & EPx running NetWare 3.1x or 4.0x

NetWare is able to automatically recognise greater than 16Mb of RAM in FT//ex and EPx, however the machine needs to be correctly configured for NetWare to do so.

To enable this facility the machine needs to be configured in the Advance Configuration option on the EISA setup disk by following the instructions below.

Once the configuration program has loaded go to the section EISA SYSTEM.

Listed in this section will be a list of available memory regions.

i.e. 16-64 Ctrl -R add Memory
64-128 Ctrl -R add Memory
128-192 Ctrl -R add Memory

These regions show memory above 16Mb in 64Mb blocks.

To configure extra memory go to the correct section (Amount of memory above 16) and type “Ctrl R”. This will bring up a separate menu with a box labelled SIZE. In this box, type in the amount of memory you are adding.

If your extra memory spans multiple memory regions, enter a value in all the appropriate memory sections.

i.e. Adding an extra 32Mb
Type “Ctrl R” at the 16-64 add memory section
Type “32M” at the size sub menu
Adding an extra 80Mb
Type “Ctrl R” at the 16-64 add memory section
Type “64M” at the size sub menu
Type “Ctrl R” at the 64-128 add memory section
Type “16M” at the size sub menu

After the above, save the configuration and Reboot the machine. NetWare will then auto register the memory correctly.
