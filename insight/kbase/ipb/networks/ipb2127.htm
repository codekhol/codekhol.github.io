Updated NetWare Files

We have recently received a number of updated files from Novell. All the files mentioned below can be found in File Area 37 of the Apricot Technical Support BBS.

NCP Packet Signature Security Fix

In response to the recently highlighted loophole in standard NetWare 3.11 security, Novell have released a full fix for the problem. The fix comes in the form of a number of updated files that replace existing files and utilities on the server, and any DOS, Windows or OS/2 workstations connected to the server. The fix has full documentation supplied with it. This version is a bug fix release that replaces the files supplied by Novell in December 1992.

The fix files for the NCP Packet Signature fix are......:-

SECDOC.ZIP The NCP Fix documentation
SECSYS.EXE The NCP Fix files for SYS:SYSTEM
SECDOS.EXE The NCP Fix files for DOS/Windows workstations
SECOS2.EXE The NCP Fix files for OS/2 workstations
SECUT1.EXE The NCP Fix files for SYS:PUBLIC (1 of 3)
SECUT2.EXE The NCP Fix files for SYS:PUBLIC (2 of 3)
SECUT3.EXE The NCP Fix files for SYS:PUBLIC (3 of 3)
SECPRN.EXE The NCP Fix files for NetWare Printing

NetWare 3.11 Patch Kit 13

Novell have recently updated the NetWare 3.11 Patch Kit. 311PTD.ZIP contains 48 dynamic patches, a new release of VREPAIR.NLM, and 2 'static' patches (patches that are applied to SERVER.EXE). New patches are....

DEVRQFIX.NLM: Fixes an Abend GPE error in the ADISK process
GVIWHFIX.ZIP: Fixes an NCP Get Volume Info With Handle call
PPCFIX.NLM: Fixes a problem when unloading an NLM abends the server
REMFILFX: Fixes the 'Abend RemoveFile called with invalid file handle' error
ROUTERFX.NLM: Fixes a NetWare 3.11 server RIP and SAP packet problem
RPLENFX.NLM: Fixes a problem where the server leaves files open
SPXLISFX.NLM Fixes a problem where SPX is incorrectly reporting listen event control blocks

Also changed are updated versions of OS2OPNFX.NLM, and SPXFIX1.NLM has been removed from the kit.. 311PTD.ZIP replaces all previous 311PT?.ZIP files. Full documentation is included in the archive.

NetWare Utility Updates

Novell have released new bug-fix versions of NDIR and NCOPY. NDIR version 3.45 can be downloaded from File Area 37 of the Apricot BBS as NDR345.ZIP. NCOPY version 3.58 can be downloaded from File Area 37 of the Apricot BBS as NCY358.ZIP. Both files contain full documentation explaining the changes made.
