Network Drivers and the Apricot On-Board Ethernet Adapter

The purpose of this bulletin is to provide information on the version of Network Driver(s)/Ethernet PALs required for a multitude of Network Operating Systems running on Apricot Qi 600 series or FT Server machines using the Apricot on-board Ethernet adapter.

The first table will describe those machines which are commonly used as Network File Servers, and the Ethernet PALs required to be installed to avoid the possibility of various errors occurring on the servers under high network traffic situations.

Typically, these errors will be either 'Abend: NMI parity error' for NetWare 286, or 'The system detected an interrupt that cannot be masked: TRAP 0002' for LAN Manager.

Machine Type Motherboard Rev. PAL Serial Number Used
VX FT 486 / 386 G NGGL 185F Part No. 12165031
VX FT 386 E D592 AG1VI Part No. 12160431
Qi 600 series G NGGL 185F Part No. 12165031
Qi 600 series E D592 AG1VI Part No. 12160431

The PAL for a Rev E motherboard is located on a daughterboard module that can be found at the rear of the board near the keyboard and mouse sockets.

The PAL for a Rev G motherboard is located at IC185 at the rear of the board approximately 4cm to the left of the VGA socket. Note however that on an FT, this is underneath the expansion backplane, and as such should only be checked by an FT trained engineer.

To correctly support the above PAL and machine combinations, both the Server and Workstation on-board Ethernet device drivers have had to be re-designed. The table below lists the version numbers of these new drivers along with the Network Operating System used.

All the above network drivers are available for download from the Apricot Technical Support Bulletin Board. Note however that drivers that are a part of a chargeable Server Environment Pack (the Novell Server Drivers) will require a password before the updated drivers can be downloaded, and that password will only be issued on proof of purchase of an existing Server Environment Pack.
