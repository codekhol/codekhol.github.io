Upgrading to Windows for Workgroups 3.11 with Apricot On-board Ethernet i82596

The following configurations are detailed below:

Windows for Workgroups, NetWare, i82596 & ODI
Windows for Workgroups, i82596 & NDIS2
Windows for Workgroups, NetWare, i82596 & NDIS2

A protected mode NDIS3 driver for the Intel 82596 On-Board Ethernet (OBE) will be available shortly, but only for NetBEUI based systems. Protected mode drivers do not currently support other protocol types.

Windows for Workgroups, NetWare, i82596 & ODI

This configuration requires the latest Windows/NetWare and ODI drivers which, at the time of writing, are found in DOSUP8.EXE and WINUP8.EXE and may be downloaded from Area 37 of Apricot Insight BBS.

Remove all network drivers from CONFIG.SYS and AUTOEXEC.BAT and delete NET.CFG from C:\WINDOWS if it exists.

Install Windows without hardware detection by typing SETUP/i. This will prevent Windows trying to identify the OBE. Proceed with the installation until the Network Setup window appears. Select CONTINUE to install Windows without a network adapter.

Continue as prompted and select RETURN to MS-DOS to return to the DOS prompt without rebooting.

Edit AUTOEXEC.BAT and add the following lines somewhere after the NET START statement:

C:\WINDOWS\LSL
C:\WINDOWS\82596ODI
C:\WINDOWS\IPXODI
C:\WINDOWS\NETX

Make sure these files are located in C:\WINDOWS then reboot the system. Ignore the message which reports that a NET.CFG was not found.

Load Windows. Run Network Setup from the Network window, select the Networks icon, and choose Install Microsoft Windows Network and OTHER network. Scroll through the list, choose Novell NetWare (Workstation Shell 3.x) and continue.

Next, select Sharing... to choose sharing options for files and printers as required and continue.

Select Drivers... and Add Adapter. Scroll through the list and select IPXODI Support Driver (Ethernet), then CLOSE to accept and continue setup. When prompted for NETWARE.DRV , place the disk with unpacked WINUP8.EXE files into the floppy drive and continue.

NET.CFG will be created. AUTOEXEC.BAT, CONFIG.SYS, SYSTEM.INI and WIN.INI will be modified. From this point, do not restart Windows, but continue and exit to DOS in the normal way.

Edit NET.CFG in C:\WINDOWS and add the following parts written in italics:

Link Driver 82596ODI
Frame Ethernet_802.3
Frame Ethernet_II
Frame Ethernet_802.2
Frame Ethernet_SNAP
Protocol IPX 0 Ethernet_802.3

It may also be necessary to edit CONFIG.SYS and reset the LASTDRIVE statement to its original setting.

Windows for Workgroups, i82596 and NDIS2

Firstly, remove all network drivers from CONFIG.SYS and AUTOEXEC.BAT.

Install Windows without hardware detection by typing SETUP/i. This will prevent Windows trying to identify the OBE. Proceed with the installation until the Network Setup window appears. Select Networks... and Install Microsoft Windows Network with No Additional Network.

Continue and select SHARING... to choose the required sharing options and continue.

Select DRIVERS... and Add Adapter... Choose Unlisted or Updated Network Adapter and insert the Apricot Drivers disk into drive A:. Select Apricot 596 Ethernet and continue.

When prompted, do not restart, but Exit to MS-DOS and add the following lines to CONFIG.SYS after the line DEVICE=C:\WINDOWS\IFSHLP.SYS.

device=c:\windows\protman.dos /i:c:\windows
DEVICE=C:\WINDOWS\ETH596.DOS
DEVICE=C:\WINDOWS\NDISHLP.SYS

Windows for Workgroups, NetWare, i82596 & NDIS2

Firstly, remove all network drivers from CONFIG.SYS and AUTOEXEC.BAT.

Install Windows without hardware detection by typing SETUP/i. This will prevent Windows trying to identify the OBE. Proceed with the installation until the Network Setup window appears. Select Networks... and Install Microsoft Windows Network with Other network. Scroll through the list, select Novell NetWare (Workstation Shell 3.x) then continue and choose IPX.COM as the type of driver.

Continue and select SHARING... to choose the required sharing options and continue.

Select DRIVERS... and Add Adapter... Choose Unlisted or Updated Network Adapter and insert the Apricot Drivers disk into drive A:. Select Apricot 596 Ethernet, then change the protocols so that the first and default protocol is Microsoft NetBEUI, and the second is IPX/SPX Compatible Transport.

Continue installation until prompted for NETWARE.DRV. Place the disk with unpacked WINUP8.EXE files into the floppy drive and continue.

When prompted, do not restart, but Exit to MS-DOS and add the following lines to CONFIG.SYS after the line device=c:\windows\ifshlp.sys.

device=c:\windows\protman.dos /i:c:\windows
DEVICE=C:\WINDOWS\ETH596.DOS
DEVICE=C:\WINDOWS\MSIPX.SYS
DEVICE=C:\WINDOWS\NDISHLP.SYS

You may also wish to reset the LASTDRIVE statement to its original setting. Add the following lines to AUTOEXEC.BAT after the NET START statement:

msipx
NETX

and copy MSIPX.COM and MSIPX.SYS to C:\WINDOWS. If you do not have a copy of these files, they may be unpacked from MSIPX.ZIP, which may be downloaded from Area 37 of Apricot Insight BBS.

Edit PROTOCOL.INI in C:\WINDOWS and add the following italicised lines:

[network.setup]
version=0x3110
netcard=apr$596,1,APR$596,1
transport=ms$nwlink,NWLINK
transport=ms$ndishlp,MS$NDISHLP
transport=ms$netbeui,NETBEUI
transport=ms$ms$ipx,MS$IPX
lana0=apr$596,1,ms$netbeui
lana1=apr$596,1,ms$nwlink
lana2=apr$596,1,ms$ndishlp
lana3=apr$596,1,ms$ms$ipx
[protman]
DriverName=PROTMAN$
PRIORITY=MS$NETBEUI

[APR$596]
DriverName=ETH596$

[NWLINK]
BINDINGS=APR$596

[MS$NDISHLP]
DriverName=ndishlp$
BINDINGS=APR$596

[MS$NETBEUI]
DriverName=netbeui$
SESSIONS=10
NCBS=12
BINDINGS=APR$596
LANABASE=0

[MS$IPX]
DriverName=IPX$
MediaType=Novell/Ethernet
BINDINGS=APR$596
