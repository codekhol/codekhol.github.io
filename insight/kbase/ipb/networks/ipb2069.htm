Novell NetWare 2.x/386 Workstation Drivers and Shells

See also IPB 2114

Novell have recently issued a number of updates to the workstation shell software. Apricot and Western Digital have also issued updated workstation drivers. As the correct combination of shells and drivers is needed for Novell (especially under NetWare 386), the current versions of all component parts will be listed below. We would recommend that ALL out-of-date versions be upgraded to the versions listed below.

1) Novell shell files (issued by Novell directly)
NET2.COM Version 2.15 rev C Last shipped with NetWare 286 V2.15
NET3.COM Version 3.02, 6/2/91 All the 3.02 shells were recently
NET4.COM Version 3.02, 6/2/91 (26/3/91) released by Novell as an
XMSNET3.EXE Version 3.02, 6/2/91 upgrade pack. The files are available
XMSNET4.EXE Version 3.02, 6/2/91 through your NetWare dealer, from
EMSNET3.EXE Version 3.02, 6/2/91 NetWire, or from the Apricot Tech
EMSNET4.EXE Version 3.02, 6/2/91 Support Bulletin Board.
IPX.OBJ Version 3.02 rev A Availability as per the 3.02 shells

2) On-board Ethernet Workstation drivers (issued by Apricot directly)
82592SH.OBJ Version 1.07 To be included on the Network Drivers Disk
82592.LAN (29/4/91) Release 6 (also on the Bulletin Board)

3) Western Digital Ethernet Workstation Drivers (issued by Western Digital via Apricot)
WDPLUSSH.OBJ Version 3.10 To be included on the next WD drivers
WDPLUSSH.LAN Version 3.10 disk (also on the Bulletin Board)

Note: Old WD8003E cards (jumper configured) will require:-
WDPLUSSH.OBJ Version 3.07 Available for download from the
WDPLUSSH.LAN Version 3.07 Apricot Bulletin Board.

Note: All the above mentioned files can be found in File Area 37 on the Apricot Bulletin Board.

This information supersedes that quoted in IPB 2034, IPB 2052 & IPB 2066.
