Novell NetWare 3.12 and the Intel E100B LAN Driver

We have had a number of calls recently whereby the Intel Pro/100B LAN drivers (E100B.LAN) hangs or produces an error when loaded on a ‘out-of-the-box’ NetWare 3.12 installation.

This problem occurs because the E100B driver requires the installation of a Novell patch before it will operate correctly. The patch is a part of 312PTD.EXE, but will need to be installed manually. Steps for doing so are as follows (and assume that 312PTD.EXE is the latest available patch kit – where there is a later version, use that instead);-

  1. Ensure you have a copy of 312PTD.EXE
  2. Unpack 312PTD.EXE
  3. Once unpacked, go to the 312PTD\NATIVE\LOADER directory
  4. Copy LSWAP.EXE and LOADER.EXE from that directory to the location where your SERVER.EXE file is located (usually C:\SERVER.312)
  5. With all the files in the same directory, run

    6. LSWAP LOADER.EXE SERVER.EXE
  6. Re-start the server and continue the installation.

As soon as possible, ensure that all the latest NetWare 3.12 patches and updates are applied to the server (these can all be found in 312UP.EXE from file area 37 of the Technical Support BBS/Web site
