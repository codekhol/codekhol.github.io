FTs BIOS Revision Summary (Rev 1.1.10)

Release Date Changes
1.02.23 4/3/92 Security adds international keyboard support.

AOX, DEC Ethernet card fixes.

Fixes 1791 error in OS/2 after pressing CTRL-ALT-NUM-NUM.

OS/2 serial printing handshaking fixed.

Error 1791 after warm boot with 2 RLL drives

Incorrect polarity of video sync signals causing wrong picture size in mode 79 on the Tatung monitor.

DAC palettes different between mode 12 and 72 (red colours).

ESDI Petrel BIOS - OS/2 HPFS formatting, non unique error codes.

Support for PC225 CPU modules (486SX/25, DX/33 and DX/33+485).

Added Adaptec/IBM SCSI slowdown (abort able by shift-F12).

Support for Qi and Titan boxes (signon, slot numbers and front panel type), based on motherboard switch settings -

SW4-8 SW4-6 BOX TYPE SIGNON

ON ON TITAN FTs 486 Series Computer

ON OFF PANDORA FTe 486 Series Computer

OFF OFF Qi Qi 486 Series Computer

OFF ON RESERVED Qi 486 Series Computer

Made channel reset signal and beep routine timing independent of CPU speed.

Moved 'Port E2' set-up to before ROM checksum test (fixes Rev F logic problem which caused early part of POST to run slowly).

Moved Prom board test/jump to before enabling 486 cache.
1.02.24 5/3/92 Support for 4MB floppy disk drives on revision F machines. Drive 0 only.

Always enable ROM shadowing unless a reference disk with feature level 3 or greater is used (which does not have the option of disabling shadowing).
1.02.25 23/2/93 IBM SCSI fix, the Apricot Extended CMOS information has been moved from 0500h to 0300h and the ABIOS DMA table entry has been changed from 0FFFFh to 000005h.

OS/2 1.x LAN Man 2.1 Remote Access Service now works.

OS/2 1.x task switching 'lock up' has been cured. (Note that the new MK/42 1.20+ keyboard controller is required)

The Petrel/ESDI BIOS has been relocated to E000:C000 from E000:8000 to make room for a larger ABIOS module.
1-02-26d
BETA		 1/3/94 Windows NT hanging with Petrel card has been fixed.

OS/2 2.1 with 16Mb onboard + adapter memory problem has been fixed.

Serial printing problem with OS/2 1.3 has been fixed.

Reset during POST just after Ctl-Alt-Del problem has been fixed.

Windows NT with monochrome monitor 'Install' hang has been fixed.
1-02-26e
BETA		 18/3/94 Windows NT hanging with Petrel card has been fixed.

OS/2 2.1 with 16Mb onboard + adapter memory problem has been fixed.

Serial printing problem with OS/2 1.3 has been fixed.

Reset during POST just after Ctl-Alt-Del problem has been fixed.

Windows NT with monochrome monitor 'Install' hang has been fixed.

Extended Memory Size is now reported correctly.
1-02-26 18/3/94 This release is functionally unchanged from version 1-02-26e BETA.
