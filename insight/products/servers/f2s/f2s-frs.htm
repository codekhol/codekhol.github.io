1-04-02
BETA		 28/9/93 Specific to Pentium
Correct Pentium-60 CPU clock speed display on signon.

The following features are added for Pentium, BIOS Caching, Memory parity error checking, SLIK data bus parity error checking.

BIOS caching is disabled, if ROM is not shadowed.

Changes for Pentium and '486
Support RLL controllers and drives.

Chipset sub-address 01Bh register can be changed through XCMOS.

Hanging when floppy disabled by security has been fixed.

IBM HDU start-up delay now controlled through XCMOS.

Fixed UNIX hang when Front Panel message received but no driver loaded.

Fixed Rev B I/O board securityset-up problem.

Fixed Cirrus 'noise' at 800 x 600 x 64 K in Windows.

Support for BIOS INT 015h, function 0C9h (Processor ID) added.

Fixed video card problem with hanging after reference, without video ROM signature. Eg. Matrox.
