FT//s (Pentium Panther II) Motherboard (Rev 1.4.2)

 


RAM Configuration

Pentium Based Processor Board

256MB Maximum. Eight 36 bit SIMM sockets. Minimum system configuration = 16MB. 32MB SIMMs are industry standard and are not the same as those used in the XEN-LS II/PC.

Memory is configured as 128 bit wide, thus SIMMs must be installed in two groups of four, as follows

MM1, 3, 5, 7
MM2, 4, 6, 8

e.g. 16MB = 4 x 4MB SIMMs
