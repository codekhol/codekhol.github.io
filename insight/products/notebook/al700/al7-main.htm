AL700 / AL720 Notebooks

Product Code Prefix: AG

 Features (options for AL700/AL720 are marked 1)
  • Pentium mobile processor
    • 233MHz (AL700)
    • 266/300MHz Pentium II (AL720)
  • 48Mb SDRAM (expandable to 144Mb)
  • 512Kb level 2 cache
  • 2Mb/4Mb1 video RAM
  • 3.5" 1.44Mb FDD (Floppy Disk Drive)
  • 20x/24x1 (max) CD-ROM drive
  • 2.5" 3.2Gb/4.0Gb1 HDD
  • PC Card (PCMCIA) Type II/Type III expansion slot
  • 13.3"/14.2"1 Thin Film Transistor (TFT) colour LCD display
  • Infra-red port (IrDA 1.1 compliant)
  • Lithium ion battery
