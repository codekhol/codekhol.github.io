VS660 'Thor' ROM BIOS Summary (Rev 1.10.1)

About This Release

  • BIOS 1.00.02.CN0 THOR.
  • S3 Trio64 V+ video. Video BIOS version 1.00-09-09.
  • Crystal Audio code update.

Features / Errata Fixed In This Release

Release 2.0

  • Update the S3 video version to 1.00-09-09.
  • Update the crystal firmware. Crystal update moved to Plug n Play code.
  • Token ring remote boot problem fixed.
  • CMOS corruption problem is fixed.
  • Fixed the problem where the numeric keys did not work for date update in 'SETUP'.
  • The ATA/ATAPI drive detection code is updated.
  • Hardisk Pre-delay setup option added in the setup.
  • Fixed the 306B com port programming problem.
  • CD-ROM is the first boot device option only.
  • Gate-A20 race condition fixed.
  • Int-15 function Dah subfunction 8Ch carry flag fix.
  • CPU Autohalt fix.
  • APM serial port mouse detection fixed.

Release 1.0

  • Fixed problem where a non-system disk would hang the system during BOOT when a CD-ROM was ahead of the HDD in the boot sequence.
  • CD-ROM is only selectable as a bot device in the first boot device list.
  • Fixed CPU AutoHalt feature, results in about 5 additional watts of power being saved during a low power state.
  • Updated the S3 Trio64 V+ to version 1.00-09-00.
  • Fixed password problem where back spacing would not erase the last character.

Beta 1.02

  • Video DDC fixed.
  • Game port fixed.

Beta 1.01

  • None.

Beta 1.01

  • Intel Corporation's Phase III - BIOS™.
  • Intel Corporation's POSThaste™.
  • Intel Corporation's Product Information Storage and Retrieval Revision 1.1 (February 6, 1995).
  • No Recovery BIOS.
  • Supports Boot With PnP OS (allows Plug and Play Operating Systems to configure all non-bootable ISA Plug and Play devices).
  • Plug and Play BIOS Specification Version 1.0A (May 5, 1994) compliant.
  • Plug and Play ISA Specification Version 1.0A (May 5, 1994) compliant/support.
  • PCI BIOS Specification 2.1 compliant.
  • PCI to PCI Bridge Architecture Specification Revision 1.0 (April 5, 1994) compliant.
  • Supporting PCI Expansion Bridges, Tim Mostad, PCI SIG Technical Marketing (December 19, 1994) compliant.
  • Extended System Configuration Data (ESCD) Specification Version 1.03 (November 30, 1994) compliant.
  • Advanced Power Management (APM) BIOS Interface Specification Revision 1.1 (September 1993) compliant (also Revision 1.0 compliant).
  • ATA (IDE Hard Disk) interface support (Mode 4 and below timing support).
  • ATAPI (IDE, CD-ROM, TAPE, etc.) interface support.
  • "El Torito" Bootable CD-ROM Format Specification Version 1.0 (July 8, 1994) compliant.
  • Keyboard / Mouse Swap support.
  • National Semiconductor PC87306B Super I/O support.
  • AMI Keyboard Controller support.
  • Intel Triton Chipset support.
  • Pentium Overdrive Ready (P55CT Socket 7) support.
  • S3 Trio64 V+ Video BIOS version 1.00-05-00.
