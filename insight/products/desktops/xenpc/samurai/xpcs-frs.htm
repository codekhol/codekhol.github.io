XEN-PC (Samurai) ROM BIOS Release Summary (Rev 1.6.2)

Release Date Changes
2-00-02
BETA		 18/3/93 Inclusion of the Video BIOS in the 27C010 ROM, removing the need for the optional socket to be populated with a Cirrus Logic BIOS.

Setup code now gets uncompressed at address E400.

Quantum drives equipped with firmware 4.40 now work correctly as a master with any slave.

Fixes for all Maxtor MXT540A-related problems [please note the "Growing Error List" problem is not a BIOS issue, but a Maxtor firmware issue].

Quantum ELS170AT and Maxtor LXT535 are now supported.

The Ethernet address is now displayed when booting in Text Mode.

Setup can now be disabled with the external LockSet utility [yet to be released].

Different signons have been implemented for the PC variants.

The Remote Boot ROM socket now works.
2-00-03
BETA		 6/4/93 Occasionally the floppy drive would not be recognised at power on, this has now been fixed.

Machines used with monochrome monitors would hang if the BIOS called security for the SIN number. This has been fixed.

The CMOS defaults, used following a 'bad' CMOS detection, are now correct.

Power on sound has been re-included.

Support has been added for the Quantum LPS525AT drive.

Different signon strings are now supported for PC variants.

Monitor types can now be setup, this includes SVGA, VGA/EVGA & EVGA (Hi refresh), the help strings in setup now indicate the refresh rates used by each mode.
2-00-04
BETA		 9/4/93 CTRL ALT 'S' no longer hangs system.

3 mode floppy now supported correctly.

No more screen 'glitches' when setting up user defined fonts.

Video Mode 13h no longer allows the palette to be altered.

Version and date strings have been updated.
2-01-01
BETA		 4/5/93 'Print Screen' is now supported
2-01 4/5/93 'Print Screen' is now supported
2-02 13/5/93 A fix has been provided for various video problems resulting from the Cirrus 5422 telling the outside world it is a 5424.
2-03-02
BETA		 15/7/93 Some adapter cards did not operate correctly, this was due to the ISA bus using de coupled refresh mode. It now uses synchronous refresh mode, but as this may impact performance it is only used when the user selects 'slow' bus speed.

Various 'mad mouse' fixes have been implemented.

Various graphical boot 'POST' messages have been altered.

The memory test is now performed on graphical boot as well as text boot.
2-03-03
BETA		 19/8/93 Quantum 240 drives did not work when used as a slave (second) unit. This has been fixed.

Specialix cards would cause memory failures if addressed at 15Mb with a memory hole enabled. This has been fixed by changing the granularity of the memory test logic.

The latest video BIOS code has been included.

Date and version have been changed.
2-03 16/9/93 Bug Fixes.
WLOC Controller fix has been included. This is to change from Asynchronous to synchronous, this is instigated by changing the bus speed to SLOW.

Mad Mouse fixes.

Problem with Quantum LPS240AT drive when used as a slave not been found now fixed. A delay determined by a CMOS location (66h) is executed before looking for the drive in order for it to become ready.

Default delay = 1.4 ses.

Specialix controller and Memory hole at 15Mb no longer reports Memory Errors.

Memory test now executed for graphical boot.

All writes to PORT 680h during POST now removed.

Other Remote boot ROM now works correctly.

LOCKSET.EXE now works on XEN-PC.

Problem with multiple 'Fixed disk 1 drive failure' messages when removing SLAVE drives has been resolved.

Video Changes.
Video BIOS now uses correct VESA monitor timings.

MCLK now 45 MHz and not 50 MHz.

Problem with Windows 800 x 600 driver now fixed.

Programming of SR16 has been changed to take into account MCLK being 45MHz.

Cosmetic Changes.
Graphical Boot Copyright message now same as Text Boot Copyright message.

POST messages now reported during Graphical Boot in the form of 'Windows' style error-boxes.
2-04-01
BETA		 12/10/93 The Cirrus video BIOS 1-03 has been implemented to support the 5428 part.

User defined hard disks are now supported.

Support is included for E/F and F2 motherboards (The 'help' text for CD-ROM disable is different).

Graphical boot reports the correct VGA controller part, eg. 5422, 5426 or 5428.
2-04-02
BETA		 5/11/93 Support for drive type 27 Quantum LPS 270A and drive type 28 Quantum LPS 540A has been added.
2-04-03
BETA		 22/11/93 Support for Quantum 170 & 340 Mb drives has been added.

The text messages "User defined hard disks" and "Capacity in Mbytes" have been changed to "User-defined hard disks" and Capacity (MB)".
2-04-04
BETA		 25/11/93 The CL5428 video controller is now detected correctly.
2-04 8/12/93 This is the release version of the previous 2-04-04 BETA and is functionally unchanged
2-05 3/8/94 VLSI revision 'FC3' memory controller silicon is now supported.
2-06-01
BETA		 04/09/97 This BIOS has been released to provide year 2000 compliance.
