Checksum		 29/11/94

7400		 Fixed a bug which occurred if the motherboard is not fitted with the M5213 Local bus IDE part, but the backplane has a M1445 - ‘SETUP’ would allow ‘Hard disk PIO mode 3’ and ‘Hard disk 32 bit access’ to be selected. This should have no effect on us, as we always have the M5213 part fitted and not the backplane M1445.

The retry count for the VL-PCI bridge chip has been increased from 16 to 64. This should have no effect on us, as we never use that part.
R01-F4
BETA		 21/4/95 The Cirrus Logic video BIOS has been upgraded to V. 1-22.

Systems would hang if the Microsoft mouse driver V. 9-01 was installed without a mouse connected. This has now been resolved.
