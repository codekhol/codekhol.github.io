Setup Screens

This section lets you see the Setup screens for many of Mitsubishi's Apricot systems -- without you having to locate a suitable machine and boot it up. To select the Setup screens you wish to view from the following list, move the mouse pointer over the required combination of system model and motherboard type until the mouse pointer changes to a hand icon, then click on it.
